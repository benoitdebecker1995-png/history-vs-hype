# Workflow & Tools Reference

## Current Production Workflow

### 1. Topic Selection
**Tools:** VidIQ trend analysis
- Use VidIQ to identify trending topics
- Connect to "History vs Hype" format (historical myths with modern impact)
- Prioritize topics with "colonial borders still killing people" angle
- Check performance potential against past successes

**Decision Framework:**
- Modern relevance? (Current events connection)
- Evidence available? (Can it be fact-checked?)
- Visual potential? (B-roll, maps, documents available?)
- Controversy level? (Engaging but not inflammatory)

### 2. Script Development
**Tools:** VidIQ script generator + Claude Pro
- VidIQ generates initial draft
- Claude Pro for enhancement:
  - Improve hooks (first 15 seconds critical)
  - Better transitions
  - Generate multiple intro/outro options
  - Enhance storytelling flow

**Script Structure:**
- Hook (0:00-0:45): Modern consequence shown first
- Context (0:45-2:00): Why this matters now
- Historical deep-dive (2:00-8:00): The actual story
- Modern connections (8:00-10:00): Bring it back to today
- Call to action (10:00-12:00): Engagement + next steps

### 3. Fact-Checking Phase
**Tools:** Claude Pro + Web research
- Every factual claim needs 2+ sources
- Academic sources prioritized
- Flag contested information
- Document sources for citation
- See: research/fact-checking-protocol.md

**Critical Checks:**
- [ ] All numbers sourced
- [ ] Quotes verified from original
- [ ] Contested claims labeled
- [ ] Multiple perspectives included
- [ ] No logical fallacies
- [ ] Academic consensus noted

### 4. Recording
**Tools:** [User hasn't specified - likely camera/microphone setup]
- Record main talking head footage
- Multiple takes for difficult sections
- Shoot at multiple focal lengths (for cutting):
  - Wide (1.0)
  - Medium (1.2)
  - Close (1.4)
  - Very close (1.6)

### 5. Editing
**Tools:** DaVinci Resolve
- Import all footage
- Create rough cut following script timing
- Add B-roll (prioritize showing actual sources)
- Color grade for consistency
- Audio mixing and cleanup
- Export settings: [User should document their export settings]

**Editing Philosophy:**
- Show sources, not text overlays (educational integrity)
- Cut on action for smooth transitions
- Maintain pacing - educational doesn't mean slow
- Use jump cuts to maintain energy
- B-roll should illustrate, not distract

**Getting Help:**
- Upload screenshots to Claude for guidance
- Ask specific questions about effects, transitions
- Request step-by-step instructions for new techniques

### 6. B-Roll Sourcing
**Sources:**
- Public domain archives
- Creative Commons footage
- News footage (check usage rights)
- Historical photographs
- Maps and documents (show actual sources!)
- Stock footage (as last resort)

**Organization:**
- Create B-roll library by topic
- Tag footage by region/time period
- Keep sources documented for credits

### 7. Thumbnail Creation
**Tools:** Photoshop + Claude feedback
- Upload drafts to Claude for feedback
- Test multiple versions
- Ensure readable on mobile
- High contrast, clear focal point
- Text: 3-5 words maximum

**Thumbnail Strategy:**
- Show tension/conflict visually
- Use facial expressions if including person
- Bold, simple text
- Consistent branding elements
- A/B test different approaches

### 8. Publishing
**Tools:** YouTube Studio + VidIQ
- Optimize title (VidIQ suggestions)
- Write description with:
  - Summary
  - Timestamps
  - Sources cited
  - Links to related content
- Tags (10-15 relevant keywords)
- Upload thumbnail
- Set to public (or schedule)

### 9. Shorts Creation
**Tools:** VidIQ clipping tool
- Identify 30-60 second segments
- Focus on single point/fact
- Add text captions
- Vertical format optimization
- Post on off-days from long-form

## Content Calendar Planning

### Posting Schedule Options
**Option 1: Consistency Focus**
- 1 long-form video per week
- 3-4 shorts per week
- Same day/time each week

**Option 2: Quality Focus**
- 1 long-form every 10-14 days
- 1 short daily
- When quality is maintained

**Current Approach:**
- Posting when ready
- Using shorts on off days
- No fixed schedule yet

**Recommendation:** Establish consistency once workflow is optimized

### Content Batching
**Efficient Workflow:**
1. Research 3-4 topics in one session
2. Write 2-3 scripts in batch
3. Fact-check all together
4. Film multiple videos in one day
5. Edit and release on schedule

**Benefits:**
- Less context switching
- Consistent quality
- Buffer for busy weeks
- Reduced decision fatigue

## Emergency Troubleshooting

### Stuck on Topic Selection?
1. Review analytics-and-strategy.md
2. Check what performed well before
3. Look at current news for hooks
4. Use proven formula: "Colonial X still killing people"

### Script Not Working?
1. Check fact-checking-protocol.md
2. Verify all claims have sources
3. Test hook on someone unfamiliar with topic
4. Consider if too much detail = information overload

### Technical Issues?
1. Screenshot + upload to Claude
2. Specific question = specific answer
3. Search DaVinci/Photoshop forums
4. YouTube has tutorials for almost everything

### Creative Block?
1. Take break, watch similar content
2. Review successful past videos
3. Research new topic deeply (inspiration follows)
4. Remember: Education is the goal, not entertainment

## Notes for Future Self

**What's working:** Evidence-based approach, modern relevance, showing sources
**Keep doing:** Thorough fact-checking, balanced perspectives, 8-12 min length
**Experiment with:** Different editing styles, thumbnail approaches, posting times
**Don't compromise:** Historical accuracy, source transparency, balanced presentation

**Remember:** You're building credibility as an educational source. Quality > quantity, always.
