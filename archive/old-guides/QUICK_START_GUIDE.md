## 🚀 For Your Next Video

### Step 1: Writing the Script
**When you include a quote from a historical figure:**

```
Type: "quote-verifier: Check this attribution -
'Thomas Jefferson said X in 1776'"
```

**Skill will:**
- ✅ Verify if Jefferson said it
- ✅ Find specific source (document, letter, page)
- ✅ Flag if it's a common misattribution
- ✅ Provide correct citation

**Before you had:** Attribution errors like Luther/Aristotle
**Now you have:** Every quote verified with specific source

---

### Step 2: After First Draft
**When script is complete:**

```
Type: "script-fact-checker: [paste your script]"
```

**Skill will:**
- ✅ List all factual claims
- ✅ Show which lack sources
- ✅ Flag vague attributions
- ✅ Create verification checklist

**Before you had:** Unsourced claims that slip through
**Now you have:** Every claim documented before filming

---

### Step 3: Before Finalizing
**When ready to structure final script:**

```
Type: "retention-analyzer: [paste your script with timing]"
```

**Skill will:**
- ✅ Identify late payoffs (strongest evidence too late)
- ✅ Flag predictable drop points
- ✅ Suggest retention hooks
- ✅ Optimize segment length

**Before you had:** 1:11 drop-off point
**Now you have:** Optimized structure for 35-40% retention

---

### Step 4: After Filming
**When you have auto-generated subtitles:**

```
Type: "srt-corrector: [provide file path or paste SRT content]"
```

**Skill will:**
- ✅ Fix spelling (Vance not Wens)
- ✅ Fix proper nouns (Mirari Vos not Mirarivos)
- ✅ Fix capitalization
- ✅ Create clean subtitle file

**Before you had:** 20-30 minutes manual correction
**Now you have:** 2 minutes automated correction

---

### Step 5: Before Upload
**When ready to publish:**

```
Type: "youtube-optimizer: Create metadata for [video summary]

Video length: 11:29
Topic: Fact-checking Reagan's trickle-down economics
Key findings: [list 3-4 main points]
Sources: [list primary sources used]"
```

**Skill will:**
- ✅ Generate 3 title options (optimized length)
- ✅ Create complete description
- ✅ List all tags
- ✅ Create YouTube chapters
- ✅ Provide thumbnail specs
- ✅ Include upload timing

**Before you had:** 30-60 minutes creating metadata
**Now you have:** 2 minutes copy-paste ready package

## 📋 Pre-Filming Checklist (NEW)

Before you film anything:

- [ ] All quotes verified (quote-verifier: ✅)
- [ ] All claims sourced (script-fact-checker: ✅)
- [ ] Structure optimized (retention-analyzer: no warnings)
- [ ] Visual assets prepared

**Don't film until all boxes checked**

---

## 📋 Pre-Upload Checklist (NEW)

Before you publish:

- [ ] Subtitles corrected (srt-corrector: completed)
- [ ] Metadata optimized (youtube-optimizer: completed)
- [ ] Thumbnail created (per specs)
- [ ] Upload scheduled (Tue-Thu, 2-4 PM EST)

**Don't upload until all boxes checked**
