# History vs Hype - Content Archive

## Channel Overview
**Channel Name:** History vs Hype  
**Focus:** Evidence-based myth-busting about geopolitics, colonial history, and border disputes  
**Target Audience:** Males 25-44, international (UK, Germany, Canada, US)  
**Current Stats:** 37 subscribers, multiple videos with Kashmir video (166 views) and Israel/Palestine (236 views) performing well

## Content Strategy
- **Video Length:** 8-12 minutes
- **Format:** Payoff-first structure (show modern consequences in first 2 minutes before explaining historical causes)
- **Approach:** Connect historical events to current developments (2023-2025)
- **Key Pattern:** "This arbitrary line drawn 100+ years ago is still killing people"

## Workflow
1. Use VidIQ to suggest topics and generate initial scripts
2. Fact-check scripts thoroughly (see research/ folder)
3. Tweak scripts to fit personal style
4. Record and edit in DaVinci Resolve
5. Find B-roll footage
6. Publish and use VidIQ clipping tool for shorts

## Directory Structure
- **scripts/** - Completed and in-progress video scripts (45 scripts organized by topic)
- **research/** - Sources, fact-checking notes, and academic references
- **channel-data/** - Analytics, performance data, and growth strategies
- **video-ideas/** - Potential topics and research for future videos
- **guides/** - Production guides and templates
- **archive/** - Compass artifacts and older script versions
