# Pre-Filming Checklist - History vs Hype

**Purpose:** Ensure every video meets quality standards before filming

**When to Use:** Complete this checklist before you hit record. It prevents wasted time filming weak scripts.

---

## ✅ SCRIPT QUALITY

### Content Standards
- [ ] **Fact-checking complete** - Every claim verified with 2+ sources
- [ ] **Sources documented** - Full citations ready for video description
- [ ] **Contested claims labeled** - Any disputed facts clearly noted
- [ ] **No logical fallacies** - Arguments are sound and well-supported
- [ ] **Counter-evidence acknowledged** - Opposing views addressed fairly

### Structure & Retention
- [ ] **Hook strength** - Opening grabs attention in first 15 seconds
- [ ] **Payoff timing** - Strongest evidence appears by 0:45
- [ ] **Modern relevance** - Contemporary connections every 90-120 seconds
- [ ] **Retention prediction** - ≥ 40% if analyzed by structure-checker
- [ ] **Pattern interrupts** - Engagement triggers every 2-3 minutes
- [ ] **Clear conclusion** - Why accurate history matters today

### Voice & Tone
- [ ] **Natural delivery** - Sounds like you, not AI
- [ ] **Authority markers** - 8-10 per script (research shows, scholars found, etc.)
- [ ] **Academic framing** - Balanced, not political punditry
- [ ] **Filler management** - "But," "Now," etc. within budget
- [ ] **Date management** - Not overwhelming (max 3-4 dates in 2 minutes)

---

## 📹 VISUAL PLANNING

### B-Roll Checklist
- [ ] **B-roll list created** - Every evidence claim has visual support
- [ ] **Sources identified** - Know where to get each image/map/document
- [ ] **Primary documents ready** - Treaties, maps, census data prepared
- [ ] **Maps prepared** - Geographic visualizations ready
- [ ] **Copyright cleared** - All visuals are public domain or licensed
- [ ] **Talking head ratio planned** - 60-70% face, 30-40% B-roll

### Visual Strategy
- [ ] **Hybrid guide consulted** - HYBRID_TALKING_HEAD_GUIDE.md reviewed
- [ ] **Evidence vs decoration** - B-roll makes arguments stronger, not just fills space
- [ ] **Shot list created** - Know exactly what to film

## 📝 SCRIPT FORMATS

- [ ] **Final script saved** - Master version with sources and notes

## 🎬 PRODUCTION READINESS

### Technical Setup
- [ ] **Camera tested** - Working properly
- [ ] **Lighting checked** - Consistent, flattering
- [ ] **Audio tested** - Clear, no background noise
- [ ] **Autocue ready** - Script loaded, readable
- [ ] **Backup plan** - Know what to do if tech fails

### Personal Readiness
- [ ] **Script memorized** - Key points internalized, not reading robotically
- [ ] **Energy level good** - Well-rested, hydrated, focused
- [ ] **Time allocated** - Enough time for multiple takes
- [ ] **No distractions** - Phone off, interruptions prevented

---

## 🚦 QUALITY GATES

### Must Pass Before Filming:
1. **Fact-check complete** - NO EXCEPTIONS
2. **Retention prediction ≥ 40%** - If analyzed
3. **Modern relevance clear** - Know why this matters today
4. **B-roll checklist done** - Know what visuals you need

### Warning Signs to Address:
- ⚠️ Script feels rushed or incomplete
- ⚠️ Not confident in any claim's accuracy
- ⚠️ Can't explain why viewers should care
- ⚠️ Missing key visual evidence
- ⚠️ Hook feels weak or generic

**If any warning signs present:** STOP and fix before filming.

---

## 📊 CHANNEL STANDARDS

### History vs Hype Requirements:
- **Historical integrity above all** - Never compromise accuracy for views
- **Evidence over opinion** - Show primary sources, not just text
- **Modern relevance** - Connect to 2023-2025 events
- **Academic balance** - Present multiple perspectives
- **No oversimplification** - Maintain nuance while being accessible

### Performance Expectations:
- **Retention goal:** 30-35% (excellent for educational)
- **Length target:** 8-12 minutes
- **Engagement:** Trigger every 2 minutes
- **Authority:** Sound knowledgeable but accessible

---

## 🎯 FINAL CHECK

Before you press record, ask yourself:

1. **Would I cite this in an academic paper?** (Fact-checking standard)
2. **Does this grab attention in 15 seconds?** (Hook strength)
3. **Can I explain why this matters today?** (Modern relevance)
4. **Do I have visual evidence for key claims?** (B-roll readiness)
5. **Does this sound like me?** (Voice authenticity)

If you answered "no" to any question, address it before filming.

## 💡 PRO TIPS

1. **Film multiple takes** - Especially hook and conclusion
2. **Mark good takes** - Note which takes felt best
3. **Stay on camera for arguments** - B-roll is evidence, not decoration
4. **Trust your research** - Speak with authority earned through verification
5. **Natural delivery beats perfection** - Minor stumbles are fine if authentic

---

## 🔄 POST-FILMING WORKFLOW

After filming:
1. **Review footage** - Check audio, lighting, delivery
2. **Note best takes** - Document which sections are strongest
3. **Verify B-roll needs** - Confirm what visuals you need in editing
4. **Update project status** - Move to editing phase
5. **Back up footage** - Immediately save raw files

---

## ❌ RED FLAGS - DO NOT FILM IF:

- Any claim lacks 2+ sources
- You're not confident explaining the topic off-camera
- Hook doesn't grab attention immediately
- Script fails voice consistency check
- Missing critical B-roll with no alternative
- Retention analysis shows < 35% prediction
- You're tired, distracted, or not feeling it

**Quality over schedule, always.**
