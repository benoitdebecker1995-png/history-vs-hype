## âœ… THE FOUR CRITERIA TEST

**Before making a video on ANY topic, it must pass ALL FOUR tests:**

### 1. PRIMARY SOURCES TEST
**Question:** Can I show original documents/evidence ON SCREEN?

âœ… PASS Examples:
- Original treaty texts
- Court transcripts
- Propaganda posters
- Government records
- Scientific papers
- Archaeological evidence
- Photographs

âŒ FAIL Examples:

**Your Strength:** Showing the receipts. If you can't show receipts, don't make the video.

---

### 2. REAL-WORLD HARM TEST
**Question:** Does this myth cause measurable harm TODAY?

âœ… PASS Examples:
- Fuels violence (shootings, attacks, hate crimes)
- Influences discriminatory legislation
- Causes economic harm to specific groups
- Justifies ongoing oppression
- Prevents justice/reconciliation
- Distorts current policy debates
- Radicalizes people into extremism
- Prevents medical/scientific progress

âŒ FAIL Examples:
- Interesting but no modern impact
- Academic debate with no stakes
- Historical inaccuracy that doesn't matter
- Niche topic only historians care about

**Your Mission:** Combat harm. If it's not causing harm, it's not urgent enough.

---

### 3. EVIDENCE-BASED DEBUNKING TEST
**Question:** Can I present this FAIRLY and let evidence speak?

âœ… PASS Examples:
- Clear documentary evidence contradicts claim
- Scientific consensus vs propaganda
- Primary sources reveal truth
- Can present multiple perspectives
- Can acknowledge uncertainty where it exists
- Can cite credible historians/experts

âŒ FAIL Examples:
- Requires ideological framing
- Evidence is ambiguous/contested
- "My interpretation vs theirs"
- Can't find credible sources
- Too politically charged to be fair

**Your Brand:** Unbiased evidence. If you can't be fair, don't make it.

---

### 4. AUDIENCE RELEVANCE TEST
**Question:** Will educated 25-44 year olds care about this?

âœ… PASS Examples:
- Connected to current news/politics
- Explains confusing modern phenomenon
- "Why does this keep happening?"
- Trending conspiracy theory
- Major policy debate context
- Cultural/social issue with historical roots

âŒ FAIL Examples:
- Too niche/academic
- No connection to modern life
- Requires extensive background knowledge
- Not searchable on YouTube
- Only matters to specialists

**Your Audience:** Smart people seeking truth about current issues. If they wouldn't care, neither should you.

## ðŸ” TOPIC GENERATION PROCESS

**How to find topics systematically:**

### Step 1: Monitor News for Propaganda
- What historical myths are being cited in current news?
- What conspiracies are trending on social media?
- What "historical precedent" is being invoked in policy debates?
- What "always been this way" arguments are you hearing?

### Step 2: Check Search Volume
- Run through VidIQ: Is anyone searching for this?
- Look at related videos: Are they getting views?
- Check competition: Can you rank?

### Step 3: Source Availability Check
- Can you ACCESS the primary sources?
- Are they in public domain / fair use?
- Can you afford image rights if needed?
- Are archives digitized?

### Step 4: Impact Assessment
- Who is being harmed by this myth?
- What specific harms can you document?
- Will debunking actually help?
- Is timing right (news cycle, legislation, etc.)?

### Step 5: Run Through 4 Criteria
- Score it honestly
- If under 15/20, keep looking
- If 18+/20, start researching immediately

## ðŸŽ¬ FROM IDEA TO VIDEO

**Once a topic scores 15+/20:**

**Week 1: Research**
- Gather primary sources
- Read academic literature
- Document modern harm instances
- Check fact-checkers' work
- Note any uncertainties

- Hook: Modern harm (0:00-0:45)

- Evidence: Show sources (2:00-8:00)
- Evolution: How myth spread (8:00-10:00)
- Impact: Why it matters (10:00-11:30)

**Week 3: Production**
- Film with script
- Edit primary sources in
- Add citations
- Graphics for timelines/stats
- Music/pacing for retention

**Week 4: Optimize & Publish**
- VidIQ title/thumbnail test
- Write compelling description
- Time for news cycle if relevant
- Monitor performance
- Engage with comments
