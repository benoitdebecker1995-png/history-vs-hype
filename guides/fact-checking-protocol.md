# Fact-Checking Protocol for History vs Hype

## Core Principle
**Maintain historical integrity above all else.** Educational content requires verified claims from credible sources.

## Primary Source Hierarchy

### Tier 1: Most Reliable
1. **Primary documents** - original treaties, census data, government archives
2. **Peer-reviewed academic publications** - university press books, scholarly journals
3. **Expert historians specializing in the topic** - cited extensively in literature

### Tier 2: Secondary Sources
4. **Respected journalists with expertise** - Thomas de Waal, Barbara Tuchman types
5. **International organization reports** - UN, ICJ, verified NGOs
6. **Declassified government documents** - with context about bias

### Tier 3: Use with Caution
7. **Credible news sources** - for modern events, always verify with multiple sources
8. **Documentary evidence** - photographs, footage (verify authenticity and context)

### Avoid
❌ Social media claims
❌ Propaganda from involved parties (without clearly labeling as such)
❌ Secondary citations without checking original source
❌ Wikipedia (use as starting point only, verify all claims)

## VidIQ Script Fact-Checking Process

### Step 1: Identify All Factual Claims
Go through VidIQ-generated script and highlight:
- Specific numbers (casualties, populations, dates)
- Historical events and their outcomes
- Cause-and-effect relationships
- Quotes from historical figures
- Legal or treaty provisions

### Step 2: Source Each Claim
For EVERY factual claim, find:
- At least 2 independent sources
- Preferably 1 academic source
- Original source if possible (not someone citing someone)

### Step 3: Flag Contested Information
When sources disagree:
- ✓ Present multiple perspectives clearly
- ✓ Explain why numbers/interpretations differ
- ✓ Use language like "Armenian sources claim X, while Azerbaijani sources claim Y"
- ✓ State scholarly consensus if one exists
- ✗ Don't present one side's numbers as fact

### Step 4: Verify with Claude/AI
- Feed sources to AI for cross-checking
- Ask AI to identify claims needing more evidence
- Have AI check for logical fallacies or unsupported conclusions
- Request verification of statistical claims

## Red Flags to Watch For

### In Your Own Scripts
- Suspiciously precise numbers without clear source
- Absolutist language ("all historians agree," "always," "never")
- Conspiracy framing ("deliberate cover-up") without documentation
- Serious accusations (fraud, bias) without specific evidence
- Cherry-picked evidence that ignores counter-evidence

### In Sources You're Using
- Politically motivated sources without disclosure
- Citations that don't actually support the claim when checked
- Secondary sources that misrepresent primary sources
- Dated scholarship contradicted by recent research
- Sources with obvious agenda presenting one-sided narratives

## Specific Historical Topics - Key Considerations

### Nagorno-Karabakh / Armenia-Azerbaijan
- **Challenge:** Highly politicized with competing historical narratives
- **Required:** Primary sources (1823 survey, Ottoman records, Soviet archives)
- **Watch for:** Demographic numbers presented with false precision
- **Verify:** Academic fraud claims with original scholarship documenting it

### Armenian Genocide
- **Challenge:** Denial machinery creates false "controversy"
- **Required:** Rely on international historian consensus
- **Watch for:** False equivalency ("both sides" when one has evidence)
- **Verify:** Lobbying expenditure claims with official records

### Israel/Palestine
- **Challenge:** Contemporary political implications, strong partisanship
- **Required:** Multiple perspectives clearly labeled
- **Watch for:** Population statistics varying by date/boundaries used
- **Verify:** Specific historical events (evacuation orders, military operations) with archival sources

### Colonial History
- **Challenge:** Apologists vs. critics, both can cherry-pick
- **Required:** Primary colonial documents + indigenous perspectives
- **Watch for:** Minimization of atrocities OR exaggeration beyond evidence
- **Verify:** Specific policies (encomienda system) with legal documents and implementation records

## Comment Response Protocol

When someone challenges your facts:
1. **Never get defensive** - thank them for engagement
2. **Check their claim** - they might be right!
3. **If they're correct:** Acknowledge it, post correction
4. **If they're wrong:** Provide sources politely, don't be condescending
5. **If complex:** Acknowledge nuance exists, offer to research more

## Tools for Fact-Checking

### Research Databases
- Google Scholar (peer-reviewed articles)
- JSTOR (academic journals - need access)
- Archive.org (historical documents)
- National Archives (various countries)

### For Modern Claims
- ProPublica (investigative journalism)
- Bellingcat (open-source investigations)
- FactCheck.org, Snopes (for debunking common myths)

### For Primary Sources
- Library of Congress
- National Archives (US, UK, etc.)
- University repositories
- Declassified document databases

## Pre-Production Checklist

Before filming ANY video:
- [ ] Every number has a source
- [ ] Every quote is verified from original
- [ ] Contested claims are clearly labeled
- [ ] At least 2 sources for each major point
- [ ] Academic sources for historical interpretation
- [ ] Modern relevance connections are accurate
- [ ] No logical fallacies in arguments
- [ ] Bias acknowledged where it exists

## The Golden Rule
**If you can't verify it with credible sources, don't include it.**

It's better to have a shorter, fully verified video than a longer one with questionable claims that could undermine your credibility.
