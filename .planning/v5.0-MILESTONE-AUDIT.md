### Integration Issue

**`get_refresh_summary` argument type mismatch (HIGH)**

`tools/intel/refresh.py` defines `get_refresh_summary(store: KBStore)` expecting a KBStore instance, but all 3 callers pass the dict returned by `run_refresh()`:

- `.claude/commands/intel.md` lines 82-84
- `.claude/commands/research.md` lines 59-60
- `.claude/commands/script.md` lines 68-69

**Impact:** `AttributeError: 'dict' object has no attribute 'get_active_channels'` on every refresh path. Affects `/intel --refresh`, `/research --new` (when stale), `/script` (when stale).

**Fix:** Refactor `get_refresh_summary()` to accept the `run_refresh()` result dict directly (already contains all needed data), or change callers to pass `KBStore()`.
