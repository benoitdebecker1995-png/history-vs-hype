# Milestone v1.4: Learning Loop

**Status:** SHIPPED 2026-02-02
**Phases:** 19-21
**Total Plans:** 4

## Overview

Learning Loop milestone — "Based on everything — your performance, competition, skills, and constraints — here are the best topics to make next."

The goal was to build a recommendation system that suggests NEW topics ranked by fit to winning patterns × competition gap × production feasibility, excluding in-progress projects.

## Milestone Summary

**Key Decisions:**

- Conversion formula: (subscribers_gained / views) * 100
- Topic classification: Reuse TAG_VOCABULARY from patterns.py
- Angle classification: Reuse classify_angles from Phase 16
- JSON angle storage: SQLite TEXT column with JSON parsing
- Strength normalization: min(100, (category_avg / overall_avg) * 50)
- Topic matching: Word-level comparison, not substring
- Multiplier cap: Maximum 1.5x boost
- Exclusion sources: Scan _IN_PRODUCTION/ and _ARCHIVED/ folders

**Issues Resolved:**

- User's core problem: "When I ask for the next best video it recommends projects I already did work on"
- Solution: Folder scanning excludes 39 existing topics from recommendations
- Pattern matching boosts topics that fit proven success formula

**Issues Deferred:**

- None — all v1.4 requirements delivered

**Technical Debt Incurred:**

- None — clean implementation building on v1.3 infrastructure
