# Milestone v1.2: Script Quality & Discovery

**Status:** SHIPPED 2026-01-30
**Phases:** 11-14 (including 13.1)
**Total Plans:** 10

## Overview

Produce better scripts faster, and make videos people actually find. This milestone added automated script quality checking, voice pattern learning, keyword research tools, and streamlined NotebookLM workflows.

## Phases

### Phase 11: Script Quality Checkers

**Goal**: Automated quality checks for spoken-delivery scripts
**Depends on**: Nothing (first phase of v1.2)
**Plans**: 2 plans

Plans:
- [x] 11-01: Foundation infrastructure + stumble/scaffolding checkers
- [x] 11-02: Repetition detection + flow analysis checkers

**Details:**
- SCRIPT-01: Repetition detection (2-4 word phrases, rhetorical vs redundant)
- SCRIPT-02: Flow analysis (undefined terms, abrupt transitions)
- SCRIPT-03: Stumble test (sentence complexity, nested clauses)
- SCRIPT-04: Scaffolding counter (proportional thresholds)
- BaseChecker pattern established for all checkers
- CLI with `--all`, `--flow`, `--repetition`, `--stumble`, `--scaffolding` flags

### Phase 12: Voice Fingerprinting

**Goal**: Learn speech patterns from existing transcripts
**Depends on**: Phase 11 (builds on quality checker infrastructure)
**Plans**: 2 plans

Plans:
- [x] 12-01: Corpus builder and pattern extractor modules
- [x] 12-02: Pattern applier and CLI integration

**Details:**
- SCRIPT-05: Voice fingerprinting
- Word-level diff using difflib.SequenceMatcher
- Frequency-based pattern extraction (min_frequency >= 3)
- Temporal weighting with exponential decay (0.95^months)
- CLI flags: `--voice`, `--show-voice-changes`, `--rebuild-voice`
- Transform-then-check workflow (voice patterns applied before checkers)

## Milestone Summary

**Key Decisions:**
- Proportional thresholds for script checkers (0.002 per word)
- Word-level diff instead of character-level (performance)
- 6-category intent classification for history niche
- Error dict pattern (return {'error': msg} instead of exceptions)
- Haiku/Sonnet/Opus model tiering for cost optimization

**Issues Resolved:**
- Python 3.14 + spaCy compatibility documented (use Python 3.11-3.13)
- Windows console encoding handled with ASCII fallbacks
- pip install srt hang resolved with manual installation docs

**Issues Deferred:**
- Retention heatmap prediction (requires 30+ videos)
- Title A/B test manager (future v1.3+)
- NotebookLM API integration (waiting for Enterprise API)

**Technical Debt Incurred:**
- datetime.utcnow() deprecation warning in intent_mapper.py (Python 3.12+)
- Voice patterns empty until user runs --rebuild-voice
- Discovery diagnostics standalone (not auto-integrated into /analyze)
