- **Python 3.14 + spaCy compatibility** — spaCy 3.8 (Pydantic v1) incompatible with Python 3.14.2. Code is correct but untestable in Python 3.14 environment. Resolution: Use Python 3.11-3.13.

### Phase 12: Voice Fingerprinting
- **Setup gap** — User must install srt library (`pip install srt`) and run `--rebuild-voice` to populate patterns. This is expected behavior for a learning tool, not a code defect.
- **Empty patterns** — voice-patterns.json has correct structure but `videos_analyzed=0` until corpus analysis runs.

### Phase 13: Discovery Tools
- **Deprecation warning** — `datetime.utcnow()` in intent_mapper.py line 168 (deprecated in Python 3.12+). Should use `datetime.now(datetime.UTC)`. No functional impact.
- **Optional link** — metadata_checker -> intent_mapper integration not implemented (planned as future enhancement).
