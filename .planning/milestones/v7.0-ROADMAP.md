## Overview

Research what actually drives clicks/impressions on edu/history YouTube (388 videos across 8 channels), then rewrite tooling (title scorer, hook generator, metadata, greenlight) to produce output that matches proven niche patterns. ## Milestone Summary

**Key Decisions:**
- Niche benchmark is advisory, not a hard gate — own-channel score primary, niche percentile additive
- authority_challenge removed from all recommendation paths (0/42 in niche)
- versus reframed from "highest CTR" to "differentiation bet, unproven" (1/388 niche, n=2 own)
- Declarative titles established as default choice (n=139 niche, n=19 own, 3.8% CTR)
- Topic-to-hook mapping: territorial→cold_fact, ideological→myth_contradiction, untranslated→specificity_bomb
- political_fact_check remapped from specificity_bomb to myth_contradiction based on niche data
- Hook scoring runs after agent generation, not inside the agent
