## End-to-End Flow Verification

### Flow 1: Single Video Analysis

**User Action:** `/analyze VIDEO_ID`

**Execution Chain:**
1. analyze.py receives video ID
2. auth.get_authenticated_service() → OAuth authentication
3. video_report.generate_video_report() → engagement data
4. metrics.get_video_metrics() → views, watch time, likes
5. retention.get_retention_data() → retention curve
6. ctr.get_ctr_metrics() → CTR (with fallback)
7. comments.fetch_and_categorize_comments() → categorized comments
8. channel_averages.get_channel_averages() → benchmarks
9. channel_averages.compare_to_channel() → delta calculations
10. generate_lessons() → automated insights
11. save_analysis() → POST-PUBLISH-ANALYSIS.md

**Status:** ✓ COMPLETE (no breaks)

### Flow 2: Pattern Recognition

**User Action:** `/patterns`

**Execution Chain:**
1. patterns.py → collect_video_data()
2. Searches for POST-PUBLISH-ANALYSIS.md files
3. parse_analysis_file() → extracts structured data
4. auto_tag_video() → topic categorization
5. aggregate_by_topic() / aggregate_by_title_structure() / aggregate_by_thumbnail()
6. generate_topic_report() / generate_title_patterns_report() / generate_monthly_summary()
7. Saves to channel-data/patterns/

**Status:** ✓ COMPLETE (no breaks)
