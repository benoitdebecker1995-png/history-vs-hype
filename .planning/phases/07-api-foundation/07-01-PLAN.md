<task type="checkpoint:human-action" gate="blocking">
  <name>Task 2: Google Cloud Console setup</name>
  <action>
    User must complete these steps in browser (Claude cannot access Google Cloud Console):

    **Step 1: Create Google Cloud Project**
    1. Go to https://console.cloud.google.com/
    2. Click project selector (top left) > New Project
    3. Name: "history-vs-hype-analytics"
    4. Click Create
    5. Wait for project creation, then select it

    **Step 2: Enable APIs**
    1. Go to APIs & Services > Library
    2. Search "YouTube Analytics API" > Enable
    3. Search "YouTube Data API v3" > Enable

    **Step 3: Configure OAuth Consent Screen**
    1. Go to APIs & Services > OAuth consent screen
    2. Select "External" > Create
    3. Fill required fields:
       - App name: "History vs Hype Analytics"
       - User support email: [your email]
       - Developer contact email: [your email]
    4. Click Save and Continue
    5. On Scopes page, click "Add or Remove Scopes"
    6. Filter by "YouTube" and add:
       - `.../auth/yt-analytics.readonly`
       - `.../auth/youtube.readonly`
    7. Save and Continue
    8. On Test Users page, click "Add Users"
    9. Add your Google account email
    10. Save and Continue > Back to Dashboard

    **Step 4: Create OAuth Credentials**
    1. Go to APIs & Services > Credentials
    2. Click "Create Credentials" > "OAuth client ID"
    3. Application type: "Desktop app"
    4. Name: "History vs Hype CLI"
    5. Click Create
    6. Click "Download JSON"
    7. Save file as: tools/youtube-analytics/credentials/client_secret.json
  </action>
  <how-to-verify>
    After completing all steps:
    1. Confirm file exists: ls tools/youtube-analytics/credentials/client_secret.json
    2. Confirm it's gitignored: git status should NOT show client_secret.json
  </how-to-verify>
  <resume-signal>
