<task type="auto">
  <name>Task 1: Create metrics.py core fetcher</name>
  <files>tools/youtube-analytics/metrics.py</files>
  <action>
Create `tools/youtube-analytics/metrics.py` with:

1. **Import auth module:**
   ```python
   from auth import get_authenticated_service
   ```

2. **Main function `get_video_metrics(video_id, start_date=None, end_date=None)`:**
   - Default date range: 2020-01-01 to today (captures full video lifetime)
   - Query YouTube Analytics API with:
     - `ids='channel==MINE'`
     - `metrics='views,estimatedMinutesWatched,averageViewDuration,likes,dislikes,comments,shares,subscribersGained,subscribersLost'`
     - `dimensions='video'`
     - `filters=f'video=={video_id}'`
   - Return dict with snake_case keys for consistency

3. **Return structure:**
   ```python
   {
       "video_id": "...",
       "views": int,
       "watch_time_minutes": float,
       "avg_view_duration_seconds": int,
       "likes": int,
       "dislikes": int,
       "comments": int,
       "shares": int,
       "subscribers_gained": int,
       "subscribers_lost": int,
       "date_range": {"start": "YYYY-MM-DD", "end": "YYYY-MM-DD"},
       "fetched_at": "ISO timestamp"
   }
   ```

4. **Error handling:**
   - Wrap API call in try/except
   - Catch `googleapiclient.errors.HttpError`
   - For 400: Return `{"error": "Invalid video ID or bad request", "video_id": video_id}`
   - For 403: Return `{"error": "API quota exceeded or permission denied", "video_id": video_id}`
   - For 401: Token refresh happens automatically in auth.py (no special handling needed)
   - For no data: Return `{"error": "No data found for video", "video_id": video_id}`

5. **CLI interface:**
   ```python
   if __name__ == '__main__':
       import sys
       import json
       if len(sys.argv) < 2:
           print("Usage: python metrics.py VIDEO_ID")
           sys.exit(1)
       result = get_video_metrics(sys.argv[1])
       print(json.dumps(result, indent=2))
   ```

6. **Docstring:** Clear usage examples at module level
