  <name>Task 1: Create MetadataGenerator with title extraction and tone filter</name>
  <files>tools/production/metadata.py</files>
  <action>
Create MetadataGenerator class in tools/production/metadata.py:

1. **TitleGenerator** (internal class or methods):
   - Extract opening hook from first section (first 2-3 sentences)
   - Generate 3 title variants:
     - Variant A: Mechanism-focused ("[Topic]'s [Problem/Paradox]")
     - Variant B: Document-focused ("The [Date] [Document] That...")
     - Variant C: Paradox/status-focused (curiosity angle)
   - Apply documentary tone filter:
     - Reject if contains: SHOCKING, You won't believe, INSANE, MIND-BLOWING
     - Reject all-caps words (except acronyms like ICJ, UN, CIA)
     - Reject multiple exclamation/question marks
     - Reject "EXPOSED", "LIED", "SECRET" without evidence
   - Enforce 60-70 character limit (truncate with warning if over)
   - Return List[TitleVariant] with variant letter, title text, focus description

2. **DescriptionGenerator**:
   - First 2-3 lines: Rephrase script opening for reading (not spoken delivery)
   - KEY DOCUMENTS section: List document-type entities (treaties, rulings, memos)
   - SOURCES section: List unique academic sources from script
   - CHAPTERS section: Use SectionTiming cumulative times (MM:SS format)
   - Include [PLACEHOLDER] markers for:
     - Related video links
     - Any sections needing manual completion
   - End with hashtags from top entities

3. **TagGenerator**:
   - Primary: Entity names (places, people, documents) - deduplicated
   - Secondary: Broader topic terms derived from section headings
   - Format: Comma-separated string, 15-20 tags total
   - Priority order: Specific entities first, then general topics

4. **Tone filter constants**:
   - CLICKBAIT_PATTERNS = ['SHOCKING', 'You won\'t believe', ...]
   - ALLOWED_ACRONYMS = ['ICJ', 'UN', 'CIA', 'AU', 'EU', 'NATO', 'UNESCO', 'WHO', 'IMF', 'ICJ', 'USSR', 'UK', 'US', 'USA']
   - MAX_TITLE_LENGTH = 70
   - TARGET_TAG_COUNT = (15, 20)

Use existing Entity dataclass from entities.py. Use SectionTiming from editguide.py for timestamps.

2. Integration flow:
```python
if metadata_mode:
    from tools.production import MetadataGenerator, EditGuideGenerator, BRollGenerator
    # Parse script
    parser = ScriptParser()
    sections = parser.parse_file(script_path)
    # Extract entities
    extractor = EntityExtractor()
    entities = extractor.extract_from_sections(sections)
    # Generate shots (for section references)
    broll_gen = BRollGenerator(project_name=project_name)
    shots = broll_gen.generate(entities, sections)
    # Calculate timings
    editguide_gen = EditGuideGenerator(project_name=project_name)
    timings = editguide_gen.calculate_timing(sections, shots)
    # Generate metadata
    metadata_gen = MetadataGenerator(project_name=project_name)
    metadata = metadata_gen.generate_metadata_draft(sections, entities, timings)
    print(metadata)
```
