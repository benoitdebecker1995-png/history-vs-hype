## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Title normalization missing — near-duplicate topics in feed**
- **Found during:** Task 2 (feed verification run)
- **Issue:** Topics like "India Pakistan border" and "India-Pakistan border" treated as distinct; feed contained near-duplicate entries
- **Fix:** Added title normalization (lowercase, strip punctuation, strip common suffixes) before dedup comparison
- **Files modified:** `tools/discovery/discovery_scanner.py`
- **Verification:** Verification run showed deduplicated output
- **Committed in:** `7919656`

**2. [Rule 1 - Bug] Non-history topics passing classifier**
- **Found during:** Task 2 (feed verification run)
- **Issue:** classify_topic was returning valid scores for geography/news topics with no clear history angle, polluting the ranked feed
- **Fix:** Added non-history filter (explicit reject list for pure-news, pop-culture, and geography-only patterns)
- **Files modified:** `tools/discovery/discovery_scanner.py`
- **Verification:** Feed output no longer contains off-topic entries
- **Committed in:** `7919656`

**3. [Rule 1 - Bug] HTTP autocomplete failure on some seeds**
- **Found during:** Task 2 (verification with --verbose flag)
- **Issue:** Some seed keywords triggered HTTP 429/503 on autocomplete API; exception not caught gracefully — seed dropped silently rather than retried
- **Fix:** Added HTTP fallback path with exponential backoff (2 retries); classify_topic precision tightened to reduce false positives from partial results
- **Files modified:** `tools/discovery/discovery_scanner.py`
- **Verification:** Scan completes without silent seed drops
- **Committed in:** `5dbd988`
