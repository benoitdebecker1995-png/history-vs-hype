<task type="auto">
  <name>Task 1: Create Pattern Extractor Module</name>
  <files>tools/youtube-analytics/pattern_extractor.py</files>
  <action>
Create `tools/youtube-analytics/pattern_extractor.py` with the following functions:

**Core extraction functions:**

1. `extract_topic_ranking(videos: List[dict]) -> List[dict]`:
   - Group videos by topic_type
   - Calculate avg_conversion_rate, count, total_subs for each topic
   - Sort by avg_conversion_rate DESC
   - Return list: `[{'topic': str, 'avg_conversion': float, 'count': int, 'total_subs': int}]`

2. `extract_angle_ranking(videos: List[dict]) -> List[dict]`:
   - Group videos by angle (videos can have multiple angles)
   - Calculate avg_conversion_rate, count, total_subs for each angle
   - Sort by avg_conversion_rate DESC
   - Return list: `[{'angle': str, 'avg_conversion': float, 'count': int, 'total_subs': int}]`

3. `extract_top_converter_profile(top_videos: List[dict], n: int = 5) -> dict`:
   - Take top N converters
   - Find dominant_topic (most common topic_type)
   - Find dominant_angles (top 2 most common angles)
   - Calculate avg_duration_seconds, avg_views, avg_likes_per_view, avg_comments_per_view
   - Return profile dict with all computed attributes

4. `calculate_channel_strengths(topic_stats: dict, angle_stats: dict) -> dict`:
   - Calculate three strength scores (0-100):
     - `document_heavy`: Based on legal+historical angle performance vs average
     - `academic`: Based on ideological (fact-check) topic performance vs average
     - `legal_territorial`: Based on legal+territorial combined performance
   - Use formula: `min(100, (category_avg / overall_avg) * 50)` to normalize to 0-100

5. `generate_insights(profile: dict) -> List[str]`:
   - Generate 3-5 actionable insight strings
   - Compare top topic to average: "Territorial topics convert Nx better than average"
   - Compare top angle to average: "Legal angle correlates with X% higher conversion"
   - Note common patterns: "Top 5 converters share legal + territorial combination"

6. `extract_winning_patterns() -> dict`:
   - Orchestration function that calls all above
   - Fetches data from database via KeywordDB.get_all_video_performance()
   - Calls aggregate_by_topic, aggregate_by_angle from performance_report.py
   - Returns complete profile dict:
     ```python
     {
         'extracted_at': str,
         'videos_analyzed': int,
         'topic_ranking': List[dict],
         'angle_ranking': List[dict],
         'top_converter_profile': dict,
         'channel_strengths': dict,
         'insights': List[str]
     }
     ```

7. `generate_winning_patterns_report() -> str`:
   - Generate markdown report from extract_winning_patterns()
   - Sections: Summary, Topic Ranking, Angle Ranking, Top Converter Profile, Channel Strengths, Insights
   - Save to channel-data/patterns/WINNING-PATTERNS.md
   - Return path to saved file

**Module structure:**
- Import from database.py (KeywordDB)
- Import aggregate_by_topic, aggregate_by_angle from performance_report.py
- Use statistics.mean for averages
- Use collections.Counter for finding dominant values
- Include docstrings with examples
- Follow error dict pattern (return `{'error': msg}` on failure)

    - `python -c "from pattern_extractor import generate_winning_patterns_report; print(generate_winning_patterns_report())"` returns path string
  </verify>
