**query.py** — Query interface for the intelligence KB:
- `get_full_report(db_path: str = None) -> str`:
  - Read youtube-intelligence.md and return its full contents
  - If file doesn't exist, return message suggesting `/intel --refresh`
- `get_algo_summary(db_path: str = None) -> str`:
  - Read KBStore latest algo snapshot
  - Format: signal weights table, pipeline mechanics, longform insights
  - Concise output (under 500 words)
- `get_competitor_report(db_path: str = None) -> str`:
  - Read KBStore competitor videos
  - Format: channel table (name, last upload, avg views) + recent uploads list
  - Concise output
- `get_outlier_report(db_path: str = None) -> str`:
  - Read KBStore competitor videos where is_outlier=1
  - Include outlier analysis from pattern_analyzer
  - Format: outlier table with views, ratio, possible reasons
- `get_niche_report(db_path: str = None) -> str`:
  - Read KBStore latest niche snapshot
  - Format: duration stats, title formula distribution, trending topics
- `get_staleness_status(db_path: str = None) -> str`:
  - Return last refresh date, staleness status, next suggested refresh
- All functions return formatted Markdown strings ready for display

**intel.md** — /intel slash command:
```
/intel                          — Full intelligence report (algo + competitors + niche)
/intel --algo                   — Algorithm mechanics summary
/intel --competitors            — Competitor channel activity
/intel --outliers               — Viral/outlier videos detected
/intel --niche                  — Niche format and pattern analysis
/intel --refresh                — Force refresh (regardless of staleness)
/intel --add-channel CHANNEL_ID NAME CATEGORY — Add channel to tracking list
/intel --query "question"       — Natural language query against KB
```

Command flow:
1. Check if intel.db exists. If not, inform user and run initial refresh
2. For `--refresh`: call `run_refresh(force=True)` from tools/intel/refresh.py, display change summary
3. For `--algo/--competitors/--outliers/--niche`: call corresponding query.py function, display result
4. For no flags: display full report from youtube-intelligence.md
5. For `--add-channel`: add channel to competitor_channels.json and KBStore, confirm
6. For `--query`: read youtube-intelligence.md content, use it as context for Claude to answer the natural language question
7. Always show staleness status at bottom: "Last refreshed: {date} ({days} days ago)"

For LLM-powered algorithm synthesis: when --refresh is run, after text-analysis synthesis completes, the command should also read the scraped content and use Claude's context to produce a higher-quality synthesis, then update the algo_snapshot in KBStore and re-export the KB. This gives the "deep algorithm mechanics" the user wants.
