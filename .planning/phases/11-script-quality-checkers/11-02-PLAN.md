**repetition.py (SCRIPT-01):**
- `RepetitionChecker(BaseChecker)` class
- Uses difflib.SequenceMatcher for near-duplicate detection

**Detection logic from RESEARCH.md:**
1. Extract phrases (3+ word sequences) from each sentence
2. Find exact and near-matches (similarity >= 0.8)
3. Flag phrases appearing 3+ times
4. Distinguish rhetorical from redundant:
   - **Rhetorical patterns (allowlist - don't flag):**
     - "Not X. Not Y." (fragment emphasis)
     - Repetition within 2 sentences (potential rhetorical device)
     - Anaphora: same phrase starting consecutive sentences
   - **Redundant (flag):**
     - Same phrase scattered across script
     - Filler phrases repeated ("the thing is", "the fact that")

**Config additions:**
- `repetition_min_phrase_words`: 3
- `repetition_similarity_threshold`: 0.8
- `repetition_min_occurrences`: 3
- `repetition_rhetorical_proximity`: 2 (sentences)

**Issue format:**
```python
{
    'phrase': "here's what",
    'count': 7,
    'positions': [(line, char), ...],
    'is_rhetorical': False,
    'severity': 'warning'|'error',
    'reason': "Repeated 7 times across script"
}
```

**Stats:**
- total_phrases_analyzed
- unique_phrases
- repeated_phrases (count)
- rhetorical_detected (count)

**flow.py (SCRIPT-02):**
- `FlowChecker(BaseChecker)` class
- Uses spaCy for noun chunk extraction and sentence parsing

**Detection logic from RESEARCH.md:**

1. **Term definition detection:**
   - Extract capitalized multi-word noun phrases (potential technical terms)
   - Common patterns for definition: "X - [definition]", "X, which is", "X, meaning"
   - Flag terms used before first definition appears
   - Exclude common words (treaty, border, country, etc.) via stoplist
   - Focus on proper nouns and domain-specific terms

2. **Transition smoothness:**
   - Check for abrupt topic shifts (sentence subjects change completely)
   - Flag consecutive sentences with no connecting elements
   - Recognize transition phrases: "However", "Therefore", "As a result", "Meanwhile"

**Config additions:**
- `flow_term_min_words`: 2 (minimum words to consider a "term")
- `flow_common_terms`: ["treaty", "border", "country", "government", ...] (stoplist)
- `flow_transition_phrases`: ["however", "therefore", "as a result", "meanwhile", "consequently", ...]

**Issue format:**
```python
# Undefined term
{
    'type': 'undefined_term',
    'term': "Sykes-Picot Agreement",
    'first_use_line': 5,
    'severity': 'warning',
    'suggestion': "Define 'Sykes-Picot Agreement' before first use"
}

# Abrupt transition
{
    'type': 'abrupt_transition',
    'lines': [12, 13],
    'prev_subject': "Britain",
    'next_subject': "economic factors",
    'severity': 'info',
    'suggestion': "Consider transition phrase between these sentences"
}
```

**Accept 80% accuracy per RESEARCH.md Open Question 1:**
- High-confidence flagging for clear cases
- Lower severity ('info') for uncertain cases
- User makes final decision on flags
