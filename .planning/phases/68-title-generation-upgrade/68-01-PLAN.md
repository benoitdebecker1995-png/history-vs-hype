    Create `tools/production/title_generator.py` with two main classes:

    **TitleMaterialExtractor:**
    - `extract_from_sections(sections: list[Section]) -> dict` — Full script scan. For each section, run EntityExtractor on strip_for_teleprompter(content). Apply position weights: intro=2.0, conclusion=1.5, body=1.0. Deduplicate entities by normalized text, accumulate weighted scores.
    - `extract_from_srt(srt_path_or_text: str) -> dict` — Parse SRT (strip sequence numbers, timestamps, HTML tags per retitle_gen.py pattern lines 248-267). Apply positional heuristic: first 20% words = intro weight, last 20% = conclusion weight, middle = body weight. Run EntityExtractor on cleaned text.
    - `_extract_numbers_with_context(text: str) -> list[tuple]` — Regex for integers and percentages. Skip years (1000-2099). Include surrounding sentence as context. Per research Pattern 5.
    - `_extract_contradictions(text: str) -> list[tuple]` — Regex for negation patterns ("never...but", "contrary to", "despite", "the myth", "actually/in fact"). Return (myth_phrase, reality_phrase) tuples. Per research Pattern 6.
    - Returns dict with keys: 'numbers', 'documents', 'entities', 'contradictions' — each a list of (item, weight) tuples sorted by weight descending.

    **detect_versus_signal(entities: list, full_text: str) -> tuple[str, str, float]:**
    - Module-level function (not class method).
    - CONFLICT_MARKERS list per research Pattern 3: 'against', 'disputed', 'competed', 'rivalry', 'divided', 'vs', 'versus', 'conflict', 'war', 'battle', 'claim', 'claimed', 'rejected', 'contested', 'opposed', 'split', 'tension'.
    - Find top entity pairs (place/person/org types). Check co-occurrence within 100-word window with conflict markers. Score: hits/3.0 capped at 1.0.
    - Returns (entity_a_text, entity_b_text, signal_strength). Returns ('', '', 0.0) if no signal.

    **TitleCandidateGenerator:**
    - `generate(material: dict, versus_signal: tuple, topic_type: str = None, db_path: str = None) -> list[dict]`
    - Always generate at least 1 declarative variant using highest-weight entity/document.
    - Generate how_why variant if material has documents or contradictions.
    - Generate versus variant if signal_strength > 0 (weighted higher if signal > 0.5).
    - Generate curiosity/paradox variant if contradictions exist.
    - Use extracted numbers for specificity in titles (e.g., "The 10-15% Literacy That Changed Everything").
    - Each candidate: build title string, then call score_title(title, db_path=db_path, topic_type=topic_type). Merge score_title() result dict with the generated title.
    - Apply _apply_tone_filter from metadata.py pattern (no CLICKBAIT_PATTERNS).
    - Truncate to MAX_TITLE_LENGTH (70 chars).
