  <name>Task 3: Create citation extraction script (NBLM-03)</name>
  <files>.claude/tools/extract-citations.ps1</files>
  <action>
Create extract-citations.ps1 that parses NotebookLM output into structured claims and bibliography.

**Input format (what user pastes into a .txt file):**
```
Medieval scholars understood Earth's sphericity. [1] Bede wrote in 725 CE
that Earth is "like a ball, not a shield." [2] Sacrobosco's textbook was
used in universities for 400 years. [3]

SOURCES:
1. Lindberg, David C. "The Beginnings of Western Science" (2007), p. 147
2. Bede, "The Reckoning of Time" (Wallis translation, 1999), p. 90
3. Grant, Edward. "Planets, Stars, and Orbs" (1996), p. 265
```

**Script requirements:**

1. **Parameters:**
   - `-InputFile` (required): Path to text file with pasted NotebookLM output
   - `-OutputFile` (optional, default: "extracted-citations.md"): Output path

2. **Processing:**
   - Split content at "SOURCES:" marker
   - Parse source list into hashtable: number -> full citation
   - Extract sentences containing [N] citations using regex
   - For each sentence: clean citation markers, identify source numbers

3. **Output format:**
```markdown
# Extracted Citations

**Source file:** [input filename]
**Extracted:** [timestamp]

---

## Claims to Verify

**Claim:** [Sentence without citation markers]
- **Status:** NEEDS VERIFICATION
- **Source:** [N] [Full citation from source list]

[Repeat for each claim]

---

## Bibliography (from NotebookLM output)

[1] [Full citation 1]
[2] [Full citation 2]
...

---

## Usage

Copy verified claims to your session log, update Status:
- NEEDS VERIFICATION -> ✅ VERIFIED (confirmed in source)
- NEEDS VERIFICATION -> ⚠️ PARTIALLY VERIFIED (needs context)
- NEEDS VERIFICATION -> ❌ UNVERIFIABLE (couldn't confirm)
```

4. **Error handling:**
   - Check input file exists
   - Warn if no "SOURCES:" section found (process anyway, list citations inline)
   - Warn if citation numbers in text don't match source list
   - Handle edge cases: multiple citations in one sentence, nested quotes

5. **Output:**
   - Write to output file with UTF8 encoding
   - Print summary to console: "Extracted N claims with M unique sources to [output file]"

Include comment header explaining purpose and usage:
```powershell
# extract-citations.ps1
# Parses NotebookLM output into structured claims and bibliography
#
# Usage:
#   .\extract-citations.ps1 -InputFile "notebooklm-paste.txt"
#   .\extract-citations.ps1 -InputFile "paste.txt" -OutputFile "citations-flat-earth.md"
#
# Input: Text file with pasted NotebookLM response (include SOURCES section)
# Output: Markdown file with claims to verify and bibliography
```
