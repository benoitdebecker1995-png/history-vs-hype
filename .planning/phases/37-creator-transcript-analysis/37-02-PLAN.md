**Core functions:**

1. `load_part6_patterns() -> Dict[str, List[str]]` — Parse STYLE-GUIDE.md Part 6 to get existing 29 patterns:
   - Read STYLE-GUIDE.md, find Part 6 section
   - Extract pattern names by category (openings, transitions, evidence, rhythm, closings, additional)
   - Return dict mapping category to pattern name list
   - Purpose: Cross-reference to avoid Part 8 duplicating Part 6

2. `run_full_pipeline(transcripts_dir: Path, db_path: Path) -> Dict` — End-to-end pipeline:
   - Import transcript_analyzer.analyze_all_transcripts to get raw analyses
   - Import technique_library.TechniqueLibrary to store results
   - Call store_analysis_results on the library
   - Call synthesize_universal_patterns
   - Return summary dict

3. `synthesize_universal_patterns(lib: TechniqueLibrary) -> Dict[str, List]` — Identify universal patterns:
   - Get all techniques from DB
   - Group by category
   - Mark techniques with creator_count >= 3 as is_universal=True (update in DB)
   - Cross-reference with Part 6: if pattern matches existing Part 6 pattern, set style_guide_ref
   - Return dict of universal patterns by category with creator counts

4. `generate_part8(lib: TechniqueLibrary, part6_patterns: Dict) -> str` — Generate Part 8 markdown:
   - Header: `## Part 8: Creator Technique Library (Auto-Generated)`
   - Metadata: Last updated date, source count, transcript count
   - Sections by technique category:
     - `### 8.1 Opening Hooks` — Universal hook patterns with creator examples
     - `### 8.2 Transitions` — Universal transition patterns with examples
     - `### 8.3 Evidence Presentation` — Source integration techniques
     - `### 8.4 Pacing & Rhythm` — Structural pacing patterns
   - Each technique entry:
     - **Name** (N creators) — with Part 6 cross-ref if applicable
     - **Formula:** Template pattern
     - **When to use:** Guidance
     - **Examples:** 2-3 creator examples with attribution
   - Section at end: `### 8.5 Part 6 Cross-References` — Map Part 8 techniques to Part 6 patterns
   - If fewer than 3 universal patterns in a category, note "Insufficient cross-creator data — re-run after adding more transcripts"

5. `write_part8_to_style_guide(content: str, style_guide_path: Path = None)` — Update STYLE-GUIDE.md:
   - Default path: `.claude/REFERENCE/STYLE-GUIDE.md`
   - Find existing `## Part 8:` section boundaries
   - If exists: replace in-place (idempotent)
   - If not: insert BEFORE Part 9 (if Part 9 exists) or append at end
   - Preserve all other Parts unchanged
   - Use utf-8 encoding

**CLI with argparse:**
- `--synthesize` — Run full pipeline (analyze transcripts → store → synthesize → generate Part 8)
- `--update` — Regenerate Part 8 from existing DB data (skip re-analysis)
- `--dry-run` — Print Part 8 to stdout without writing to STYLE-GUIDE.md
- `--json` — Output raw synthesis data as JSON
- `--transcripts-dir PATH` — Override transcripts directory
- `--db PATH` — Override database path
