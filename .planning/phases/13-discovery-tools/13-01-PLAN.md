Create autocomplete.py for YouTube suggestion extraction.

**IMPORTANT:** Research shows HTTP requests to YouTube autocomplete are blocked. Must use Puppeteer with stealth plugin.

Since existing codebase uses Python, use `pyppeteer` (Python port of Puppeteer) instead of Node.js:

```python
# Installation: pip install pyppeteer pyppeteer-stealth
```

**autocomplete.py:**

`async def get_autocomplete_suggestions(seed_keyword: str, max_suggestions: int = 10) -> dict`:
- Launch headless browser with stealth settings
- Navigate to YouTube search
- Type seed keyword in search box
- Wait for autocomplete dropdown
- Extract suggestions from dropdown elements
- Close browser
- Return `{'keyword': seed_keyword, 'suggestions': [...], 'fetched_at': timestamp}` on success
- Return `{'error': msg}` on failure (rate limit, timeout, etc.)

`def extract_keywords_batch(seeds: list[str], delay_seconds: float = 2.0) -> list[dict]`:
- Process multiple seeds with random delay between (delay_seconds to delay_seconds*2)
- Implements exponential backoff on errors (1s, 2s, 4s, 8s)
- Returns list of results

`def save_to_database(suggestions_result: dict, db: KeywordDB) -> int`:
- Save suggestions to database with source='autocomplete'
- Returns count of new keywords added

**CLI interface (if __name__ == '__main__'):**
- `python autocomplete.py "dark ages myth"` - Single keyword
- `python autocomplete.py "dark ages, crusades, colonialism"` - Batch (comma-separated)
- `python autocomplete.py --seed-file seeds.txt` - File with one seed per line
- `--save` flag to persist to database
- `--json` flag for JSON output

**Rate limiting:** Minimum 2 second delay between requests, random jitter 1-3s additional

Update requirements.txt: Add pyppeteer, pyppeteer-stealth
