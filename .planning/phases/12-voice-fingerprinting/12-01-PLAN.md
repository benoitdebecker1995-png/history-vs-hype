<task type="auto">
  <name>Task 2: Create pattern extractor module</name>
  <files>
    tools/script-checkers/voice/pattern_extractor.py
  </files>
  <action>
Create `voice/pattern_extractor.py` with pattern extraction logic:

1. `extract_patterns(all_modifications: List[Dict], min_frequency: int = 3) -> Dict`:
   - Track substitutions using `collections.Counter` for `(original, modified)` tuples
   - Track deletions using `Counter` for original phrases
   - Track insertions (additions user consistently makes)
   - Filter by min_frequency (per RESEARCH.md: >=3 to distinguish patterns from ad-libs)
   - Return structured patterns dict:
     ```python
     {
       'word_substitutions': [
         {'formal': 'utilize', 'casual': 'use', 'frequency': 7, 'confidence': 'HIGH'}
       ],
       'anti_patterns': ['it should be noted that', 'in order to'],
       'additions': ['actually', 'here is the thing']  # Words user consistently adds
     }
     ```
   - Confidence levels: HIGH (freq >= 5), MEDIUM (freq >= 3)

2. `weight_by_recency(pattern_occurrences: List[Tuple], decay_factor: float = 0.95) -> Dict[str, float]`:
   - Apply exponential decay: weight = count * (decay_factor ^ months_ago)
   - Per RESEARCH.md: 0.95 decay factor = 5% decay per month
   - Return weighted scores keyed by pattern

3. `classify_pattern_type(modification: Dict) -> str`:
   - Single word -> single word: 'word_substitution'
   - Phrase -> nothing: 'deletion'
   - Nothing -> phrase: 'insertion'
   - Phrase -> different phrase: 'restructuring' (ignore for now - too variable)

4. `build_pattern_library(projects_dir: Path, output_path: Path) -> Dict`:
   - Use `find_video_pairs()` from corpus_builder
   - For each pair, run `compare_script_to_transcript()`
   - Extract video date from folder name (e.g., "1-somaliland-2025" -> 2025)
   - Collect all modifications with timestamps
   - Apply `extract_patterns()` with temporal weighting
   - Write to output_path as JSON with metadata:
     ```json
     {
       "metadata": {
         "generated": "2026-01-28T...",
         "videos_analyzed": 11,
         "video_list": ["somaliland", "belize", ...]
       },
       "patterns": { ... }
     }
     ```
   - Return the pattern dict

Handle edge cases:
- Skip videos where script is significantly different length from transcript (>50% difference = likely wrong pair)
- Log warnings for low-confidence patterns (freq < 5)
  </action>
