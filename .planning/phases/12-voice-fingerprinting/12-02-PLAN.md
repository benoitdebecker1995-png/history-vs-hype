Create `voice/pattern_applier.py` to apply learned patterns to new scripts:

1. `class VoicePatternApplier`:
   ```python
   class VoicePatternApplier:
       def __init__(self, patterns_path: Path = None):
           """Load patterns from JSON file. Default: voice-patterns.json in same directory."""
           self.patterns = {}
           self.modifications_made = []

           if patterns_path is None:
               patterns_path = Path(__file__).parent.parent / 'voice-patterns.json'

           if patterns_path.exists():
               with open(patterns_path, 'r', encoding='utf-8') as f:
                   data = json.load(f)
                   self.patterns = data.get('patterns', {})
                   self.metadata = data.get('metadata', {})
           else:
               # No patterns yet - return text unchanged
               self.patterns = {}
               self.metadata = {'videos_analyzed': 0}

       def apply(self, text: str, show_changes: bool = False) -> Tuple[str, List[Dict]]:
           """Apply patterns to text, return (modified_text, changes_list)"""
   ```

2. `apply_word_substitutions(text: str) -> Tuple[str, List[Dict]]`:
   - For each substitution pattern with confidence == 'HIGH':
     - Use regex with word boundaries: `r'\b' + re.escape(formal) + r'\b'`
     - Apply case-insensitive replacement
     - Track each replacement: `{type: 'substitution', original: formal, replacement: casual, count: N}`
   - Return modified text and list of changes

3. `remove_anti_patterns(text: str) -> Tuple[str, List[Dict]]`:
   - For each anti-pattern phrase:
     - Remove phrase (with surrounding whitespace normalization)
     - Track: `{type: 'removal', phrase: phrase, count: N}`
   - Return modified text and list of changes

4. `apply(text: str, show_changes: bool = False) -> Tuple[str, List[Dict]]`:
   - Apply word substitutions first
   - Remove anti-patterns second
   - Combine all changes into list
   - Return (final_text, changes)

5. Export function for simpler API:
   ```python
   def apply_voice_patterns(text: str, patterns_path: Path = None, show_changes: bool = False) -> Tuple[str, List[Dict]]:
       applier = VoicePatternApplier(patterns_path)
       return applier.apply(text, show_changes)
   ```

Edge cases to handle:
- If voice-patterns.json doesn't exist: return text unchanged, empty changes list
- If patterns are empty: return text unchanged, empty changes list
- If confidence is 'MEDIUM': skip (only apply HIGH confidence patterns by default)
- Preserve sentence structure when removing phrases (don't create double spaces)
