2. **Topic classification function:**
```python
# Reuse TAG_VOCABULARY from patterns.py
TAG_VOCABULARY = {
    'territorial': ['dispute', 'border', 'territory', 'claim', 'annex', ...],
    'ideological': ['myth', 'debunk', 'fact-check', 'propaganda', ...],
    'colonial': ['colonial', 'empire', 'independence', ...],
    ...
}

def classify_topic_type(title: str) -> str:
    """Classify video into primary topic type."""
    title_lower = title.lower()
    for topic, keywords in TAG_VOCABULARY.items():
        if any(kw in title_lower for kw in keywords):
            return topic
    return 'general'
```

3. **calculate_conversion_rate(views, subs_gained) -> float:**

- Return 0.0 if views == 0
