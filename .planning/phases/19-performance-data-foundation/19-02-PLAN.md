2. **aggregate_by_topic(videos: List[dict], min_count: int = 2) -> dict:**
- Group videos by topic_type
- For each group, calculate:
  - count
  - avg_conversion_rate
  - median_conversion_rate
  - avg_views
  - total_subscribers_gained
  - videos list (titles)
- Only include topics with >= min_count videos
- Return dict: topic -> stats

3. **aggregate_by_angle(videos: List[dict], min_count: int = 2) -> dict:**
- Parse angles JSON for each video (can have multiple)
- Group videos by each angle (video can appear in multiple groups)
- For each angle, calculate same stats as aggregate_by_topic
- Return dict: angle -> stats

4. **identify_top_converters(videos: List[dict], n: int = 5) -> List[dict]:**
- Sort by conversion_rate DESC
- Return top n videos with full data

5. **identify_conversion_patterns(topic_stats: dict, angle_stats: dict) -> List[str]:**
- Generate insight strings like:
  - "Territorial topics average 0.85 subs/100 views (2x higher than colonial)"
  - "Legal angle correlates with highest conversion (0.92 subs/100 views)"
- Compare best vs worst performers
- Note sample size warnings
