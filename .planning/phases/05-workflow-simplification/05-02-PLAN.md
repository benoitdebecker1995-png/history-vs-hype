  <name>Task 1: Create Smart Status Router</name>
  <files>.claude/commands/status.md</files>
  <action>
Create /status command that detects project context and suggests next action.

**Detection Logic:**

1. **Find active project:**
   - Scan video-projects/_IN_PRODUCTION/ for recent activity
   - Check git status for recently modified files
   - Identify which project user is likely working on

2. **Assess project state by checking file existence:**
   - Has `01-VERIFIED-RESEARCH.md`? Research exists
   - Has `SCRIPT.md` with content beyond placeholder? Script exists
   - Has `03-FACT-CHECK-VERIFICATION.md` with verdict? Verified
   - Has `YOUTUBE-METADATA.md`? Ready to publish

3. **Determine phase:**
   - No research file → Pre-production (suggest /research)
   - Research but no script → Production (suggest /script)
   - Script but no verification → Production (suggest /verify)
   - Verified but no metadata → Post-production (suggest /publish)
   - All complete → Ready to film or published

4. **Generate suggestion:**
   - Show current project and phase
   - Suggest specific next command with rationale
   - Include alternative actions if applicable

**Output Format:**
```
## Project Status: [Project Name]

**Phase:** [Pre-production | Production | Post-production | Complete]
**Location:** video-projects/_IN_PRODUCTION/[folder]/

### Current State
- [x] Research complete (01-VERIFIED-RESEARCH.md)
- [ ] Script written (SCRIPT.md)
- [ ] Fact-check verified (03-FACT-CHECK-VERIFICATION.md)
- [ ] Metadata ready (YOUTUBE-METADATA.md)

### Suggested Next Action
**Run:** `/script`
**Why:** Research is 90%+ verified, ready to write script from verified facts.

### Alternative Actions
- `/verify --extract` - Extract claims from external transcript
- `/sources` - Get additional source recommendations
```

Also respond to:
