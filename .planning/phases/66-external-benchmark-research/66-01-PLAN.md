**Step 3 — Outlier video identification per channel:**
For each confirmed channel, use web search or YouTube Data API (public, no auth) to:
1. Find the channel's recent 20-50 videos with view counts
2. Compute median views as baseline
3. Identify outlier videos at 3x+ the median (these are packaging successes)
4. Record for each outlier: title, view count, views/median ratio, upload date

Classify each outlier video title into pattern categories using the same names from title_scorer.py: versus, declarative, how_why, question, colon, the_x_that. If a title doesn't match any pattern, classify as "other".

Classify each outlier by topic type using existing categories: territorial, ideological, political_fact_check. Add "other" if needed.

**Step 4 — Transcript extraction for hook analysis:**
For the top 15-20 outlier videos across all channels, extract the first 2-3 sentences (the hook). Methods in priority order:
1. `youtube-transcript-api` Python library (lightest, try first)
2. `yt-dlp --write-auto-subs` (if transcript API fails)
3. Manual web search for the opening line (fallback)

Install `youtube-transcript-api` if not present: `pip install youtube-transcript-api`

For each extracted hook, classify the rhetorical move:
- `cold_fact`: Opens with a specific number, date, or measurement
- `myth_contradiction`: Opens by stating a common belief then contradicting it
- `authority_challenge`: Opens by challenging an established authority or consensus
- `specificity_bomb`: Opens with a hyper-specific detail that reframes the question

Record: channel name, video title, view count, first 2-3 sentences verbatim, rhetorical move classification, topic type.
