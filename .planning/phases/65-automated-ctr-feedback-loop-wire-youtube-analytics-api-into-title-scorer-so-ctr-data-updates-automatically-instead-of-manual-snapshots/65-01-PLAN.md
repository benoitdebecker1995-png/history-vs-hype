<objective>
Wire YouTube Analytics API CTR data into the existing ctr_tracker.py snapshot pipeline so title scorer pattern scores update automatically from real CTR data without manual intervention.

Purpose: Closes the feedback loop between published video CTR performance and the title scoring system. Currently CTR data enters the system only via manual CROSS-VIDEO-SYNTHESIS.md editing + ctr_ingest.py. After this plan, running ctr_tracker.py (manually or via Windows Task Scheduler) populates ctr_snapshots with real API CTR, and the entire title_ctr_store -> title_scorer -> greenlight chain benefits automatically.

Output: Extended ctr_tracker.py with CTR fetch, test suite, logs directory, Windows Task Scheduler setup instructions.
</objective>
