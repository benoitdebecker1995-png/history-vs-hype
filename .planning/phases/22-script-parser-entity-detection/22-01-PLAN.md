  <action>
Create `tools/production/` directory and implement ScriptParser:

**parser.py:**
1. Create Section dataclass:
   ```python
   @dataclass
   class Section:
       heading: str           # H2 heading text (or "Untitled" for headerless scripts)
       content: str           # Full text content of section
       word_count: int        # Word count for timing estimation
       start_line: int        # Line number where section starts
       section_type: str      # 'intro', 'body', 'conclusion' (inferred by position)
   ```

2. Create ScriptParser class:
   - `parse_file(path: Path) -> List[Section]` — reads file, calls parse_text
   - `parse_text(text: str) -> List[Section]` — main parsing logic
   - `_extract_frontmatter(text: str) -> Tuple[dict, str]` — strips YAML frontmatter if present
   - `_split_sections(text: str) -> List[Section]` — splits on ## headers

3. Section parsing logic:
   - Use regex `r'^## (.+)$'` with MULTILINE flag to find H2 headers
   - If no ## headers found, treat entire script as single "Untitled" section
   - Skip markdown elements: `<!-- comments -->`, triple-backtick code blocks
   - Strip B-roll annotations `[B-ROLL: ...]` and `**[ON-CAMERA]**` from word count (but keep in content)
   - Infer section_type: first section = 'intro', last section = 'conclusion', middle = 'body'

4. Word count calculation:
   - Strip markdown formatting before counting
   - Exclude B-roll markers `[...]` from count
   - Count only actual spoken words

**__init__.py:**
```python
from .parser import ScriptParser, Section
from .entities import EntityExtractor, Entity

__all__ = ['ScriptParser', 'Section', 'EntityExtractor', 'Entity']
```

<task type="auto">
  <name>Task 2: Create EntityExtractor with hybrid detection</name>
  <files>
    tools/production/entities.py
  </files>
  <action>
Implement EntityExtractor with hybrid regex + optional spaCy approach:

**entities.py:**
1. Create Entity dataclass:
   ```python
   @dataclass
   class Entity:
       text: str              # Entity text as found in script
       entity_type: str       # 'place', 'person', 'document', 'date', 'organization'
       mentions: int          # Number of times mentioned
       positions: List[int]   # Line numbers where mentioned
       normalized: str        # Lowercase, normalized form for deduplication
   ```

2. Create EntityExtractor class:
   - `__init__(self, use_spacy: bool = False)` — optional spaCy enhancement
   - `_nlp` property with lazy loading (follow flow.py pattern)
   - `extract(text: str) -> List[Entity]` — main extraction method
   - `extract_from_sections(sections: List[Section]) -> List[Entity]` — extracts and merges

3. Entity detection approach (hybrid):

   **Regex patterns (always available, no dependencies):**
   - **Documents:** Treaties, Acts, Orders, Rulings, Agreements
     - Pattern: `(the\s+)?[A-Z][A-Za-z\s]*(Treaty|Act|Order|Agreement|Accord|Convention|Protocol|Ruling|Resolution|Declaration|Memo|Constitution)\s*(of\s+\d{4})?`
     - Examples: "Somaliland Order in Council", "British Somaliland Independence Act 1960"

   - **Dates:** Specific dates with context
     - Pattern: `(January|February|...|December)\s+\d{1,2}(st|nd|rd|th)?,?\s+\d{4}`
     - Also: `\d{4}` when followed by event context (e.g., "in 1960")

   - **Places:** Capitalized geographic names
     - Use keyword list: continents, countries, regions, cities, bodies of water
     - Pattern: `(the\s+)?[A-Z][a-z]+(\s+[A-Z][a-z]+)*` + location context words (Island, Peninsula, Gulf, etc.)

   - **Organizations:** Known patterns
     - Pattern: UN, AU, ICJ, EU, NATO, plus "the [Adjective] [Noun]" where Noun is org word (Union, Court, Council, Organization)

   - **People:** Titles + Names
     - Pattern: `(Prime Minister|President|Foreign Minister|Premier|King|Queen|General)\s+[A-Z][a-z]+(\s+[A-Z][a-z]+)?`
     - Also detect raw capitalized names near title words

   **spaCy enhancement (optional, for higher accuracy):**
   - If use_spacy=True and spaCy available, use NER for:
     - PERSON, GPE (places), ORG, DATE labels
     - Merge with regex results (spaCy catches more, regex catches domain-specific)
   - Graceful fallback if spaCy not installed

4. Deduplication logic:
   - Normalize: lowercase, strip "the ", collapse whitespace
   - Merge entities with same normalized form
   - Aggregate mention counts and positions

5. Domain-specific keyword lists (follow classifiers.py pattern):
   ```python
   DOCUMENT_KEYWORDS = ['treaty', 'act', 'order', 'agreement', ...]
   PLACE_INDICATORS = ['island', 'peninsula', 'gulf', 'ocean', 'country', ...]
   ORG_KEYWORDS = ['union', 'court', 'council', 'organization', 'committee', ...]
   TITLE_KEYWORDS = ['prime minister', 'president', 'foreign minister', ...]
   ```

Handle Windows encoding issues (cp1252 console output) by using ASCII-safe output.

1. Add `if __name__ == "__main__":` block at end of parser.py:
   ```python
   if __name__ == "__main__":
       import sys
       if len(sys.argv) < 2:
           print("Usage: python parser.py <script.md>")
           sys.exit(1)

       from pathlib import Path
       from entities import EntityExtractor

       script_path = Path(sys.argv[1])
       if not script_path.exists():
           print(f"File not found: {script_path}")
           sys.exit(1)

       parser = ScriptParser()
       extractor = EntityExtractor()

       sections = parser.parse_file(script_path)

       print(f"\n=== SECTIONS ({len(sections)}) ===")
       total_words = 0
       for i, section in enumerate(sections, 1):
           print(f"{i}. {section.heading} ({section.section_type})")
           print(f"   Words: {section.word_count} | Line: {section.start_line}")
           total_words += section.word_count

       print(f"\nTotal words: {total_words}")
       print(f"Est. runtime: {total_words / 150:.1f} min @ 150 WPM")

       entities = extractor.extract_from_sections(sections)

       print(f"\n=== ENTITIES ({len(entities)}) ===")
       for etype in ['document', 'place', 'person', 'date', 'organization']:
           typed = [e for e in entities if e.entity_type == etype]
           if typed:
               print(f"\n{etype.upper()}S ({len(typed)}):")
               for e in sorted(typed, key=lambda x: -x.mentions)[:5]:
                   print(f"  - {e.text} ({e.mentions}x)")
   ```
