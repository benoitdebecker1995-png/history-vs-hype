Create `tools/production/intake_parser.py` with the following structure:

**Response type classification (regex-scoring approach per RESEARCH.md):**

```python
def classify_paste(text: str) -> dict:
    """Auto-classify pasted VidIQ/Gemini response text.
    Returns {
        'type': str,         # 'keyword_data' | 'title_suggestions' | 'thumbnail_concepts' | 'description_draft' | 'tag_set' | 'unknown'
        'confidence': float, # 0.0-1.0
        'parsed_items': list,# Extracted structured items
        'preview': str       # Short preview for user confirmation
    }
    """
```

**Detection signal patterns (5 response types):**

1. `keyword_data`: Lines with volume/competition numbers, "search volume", score patterns like "XX/100", keyword lists with metrics
   - Signals: `r'\b(?:search\s+volume|monthly\s+searches)\b'`, `r'\b\d+\s*/\s*100\b'`, `r'\bcompetition[:\s]+\d+'`, `r'\bvolume[:\s]+[\d,]+'`
   - Parser: Extract keyword + metric pairs into `[{'term': str, 'volume': str, 'competition': str, 'score': str}]`

2. `title_suggestions`: Numbered/bulleted title candidates (40-70 chars), "Title:" prefix patterns
   - Signals: `r'^[\d]+[\.\)]\s+[A-Z].{20,65}$'`, `r'\bTitle[:\s]'`, `r'^\d+\.\s+".{20,65}"'`
   - Parser: Extract title strings into `[{'title': str, 'char_count': int}]`

3. `thumbnail_concepts`: "Thumbnail", "visual", "background", color descriptors, compositional language
   - Signals: `r'\b(?:thumbnail|visual|background|composition)\b'`, `r'\b(?:left|right|split|overlay|foreground)\b'`
   - Parser: Extract concept descriptions into `[{'label': str, 'description': str}]`

4. `description_draft`: Multi-paragraph prose with YouTube-style structure, hashtag blocks
   - Signals: `r'^#{1,3}\s'` (markdown headers), multi-line prose blocks, `r'#\w+'` (hashtags), `r'subscribe|like|comment'`
   - Parser: Extract full text as single item `[{'text': str, 'word_count': int}]`

5. `tag_set`: Comma-separated short phrases, or dense list of 15+ short keyword phrases
   - Signals: High comma density, short average phrase length, 15+ items when split by comma
   - Parser: Extract tags into `[{'tag': str}]`

**Scoring:** For each type, score = (matching signals / total signals for that type). Best score wins. If best < 0.25, return 'unknown'.

**Preview generation:** For each type, build a short preview string:
- keyword_data: "8 keywords detected. First: 'gibraltar history (12,000 vol, 45 comp)'"
- title_suggestions: "5 titles detected. First: 'Spain's 300-Year Trap...'"
- thumbnail_concepts: "3 concepts detected. First: 'Cartographic Collision'"
- description_draft: "Description draft (245 words)"
- tag_set: "22 tags detected. First 3: 'gibraltar, treaty of utrecht, spain'"
