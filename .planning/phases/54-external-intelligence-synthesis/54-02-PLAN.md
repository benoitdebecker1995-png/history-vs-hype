**3 variant generation (per CONTEXT.md — exactly 3 title+thumbnail pairings):**

Each variant is a coherent title+thumbnail pairing labeled by test hypothesis:

1. **Variant A: Keyword-Optimized** — Title maximizes search volume keywords from VidIQ data. Thumbnail emphasizes discoverability (clear subject, readable text).
2. **Variant B: Curiosity Gap** — Title creates knowledge gap from Gemini creative angles. Thumbnail emphasizes intrigue (contrast, question, partial reveal).
3. **Variant C: Authority Angle** — Title establishes intellectual authority from script entities. Thumbnail emphasizes evidence (primary source, document, map).

For each variant, build title by combining weighted inputs:
- Collect title candidates from all sessions of type `title_suggestions`
- Collect keyword data from sessions of type `keyword_data`
- Get competitor outlier titles via `prompt_generator._get_competitor_context()` (import from sibling module)
- If script_path provided, use `ScriptParser` + `EntityExtractor` for internal entities
- Apply `MetadataGenerator._apply_tone_filter()` to all title candidates (filter clickbait per channel DNA)
- Score and rank title candidates for each variant type using SOURCE_WEIGHTS
- Select best title for each variant, ensuring all 3 are distinct

**Content moderation scoring (per CONTEXT.md — full metadata moderation):**

```python
MODERATION_TRIGGERS = {
    'HIGH': ['genocide', 'massacre', 'holocaust', 'nazi', 'torture', 'execution', 'ethnic cleansing', 'rape', 'terrorism'],
    'MEDIUM': ['coup', 'assassination', 'war crimes', 'atrocities', 'propaganda', 'dictator', 'apartheid', 'slavery'],
    'LOW': ['colonial', 'occupation', 'disputed', 'controversial', 'conflict', 'invasion']
}
```

For each variant's title, description text, tags, AND thumbnail concepts:
- Scan for trigger words (case-insensitive)
- Assign risk level: highest trigger found
- Generate context-aware notes (per RESEARCH.md Pitfall 4): "Contains 'Nazi' — YouTube may restrict monetization. Standard for historical/educational content."
- For HIGH-risk thumbnail concepts: "Use archival photos rather than AI-generated imagery" (per CONTEXT.md)
- Suggest safe alternatives for HIGH-risk terms in titles: e.g., "Holocaust" → keep (channel policy: explicit about sensitive topics), but note monetization risk

**Moderation is informational, NOT blocking** (per CONTEXT.md decision). HIGH = flag with note + alternatives, not prevent.

**Add new section: INTAKE PARSING (`--intake`)**

```markdown
## INTAKE PARSING (`--intake`)

Parse pasted VidIQ/Gemini responses into structured data for synthesis.

### Session Flow

1. System prompts: "Paste your VidIQ/Gemini response"
2. User pastes raw text
3. System auto-detects type (keyword data, titles, thumbnails, description, tags)
4. System shows preview: "Detected: keyword data (8 keywords). First: 'gibraltar history (12,000 vol)'. Confirm? [y/n]"
5. User confirms → saved to EXTERNAL-INTELLIGENCE.json
6. System prompts for next paste, or user types 'done'
7. On 'done': auto-runs synthesis engine

### Workflow

```python
from tools.production.intake_parser import classify_paste, save_session, load_or_create_intelligence
from tools.production.synthesis_engine import synthesize

# For each paste:
classified = classify_paste(pasted_text)
# Show preview to user, get confirmation
save_session(project_path, source='vidiq_pro_coach', classified=classified, raw_text=pasted_text)

# After 'done':
result = synthesize(project_path, script_path)
```

### Auto-Detection Types

| Type | Detected By | Example Signal |
|------|-------------|----------------|
| keyword_data | Volume/competition numbers | "search volume: 12,000" |
| title_suggestions | Numbered title lists (40-70 chars) | "1. Spain's 300-Year Trap..." |
| thumbnail_concepts | Visual/compositional language | "Split screen with map overlay" |
| description_draft | Multi-paragraph YouTube prose | Paragraphs with hashtags |
| tag_set | Comma-separated keyword phrases | "gibraltar, treaty, spain, ..." |

### Source Labeling

When prompting for paste, ask which tool the response came from:
- "vidiq_pro_coach" (Steps 1-4 from --prompts)
- "gemini" (Step 5 from --prompts)
```
