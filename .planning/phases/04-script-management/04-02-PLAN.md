# Plan 04-02: Teleprompter Export

**Phase:** 04 - Script Management
**Requirement:** SCRP-02 (Teleprompter-ready export format)

## Objective

Create a teleprompter export that strips markdown formatting and B-roll notes, producing clean readable text for PromptSmart.

## Tasks

<task id="1">
**Define export format specification**

Teleprompter output requirements:
- **Format:** Plain text or RTF
- **Font:** Large (24-32pt for readability)
- **Content:** Spoken words ONLY
- **Strip:** All markdown (`#`, `**`, `[]`, etc.)
- **Strip:** B-roll notes (`[B-ROLL: ...]`)
- **Strip:** Source citations (`[SOURCE: ...]`)
- **Strip:** Section headers that aren't spoken
- **Strip:** Metadata (target length, framing notes)
- **Keep:** Natural paragraph breaks for pacing
</task>

<task id="2">
**Create Python export script**

Create `.claude/tools/teleprompter-export.py`:
- Input: Path to SCRIPT.md
- Output: Clean text file in same directory as `SCRIPT-TELEPROMPTER.txt`
- Use regex to strip B-roll/source markers
- Remove markdown formatting
- Preserve paragraph structure

Test with project 30 Belavezha script.
</task>

<task id="3">
**Create /teleprompter command**

Create `.claude/commands/teleprompter.md`:
- Takes project path or auto-detects from context
- Runs export script
- Reports word count and estimated runtime
- Confirms output location
</task>

<task id="4">
**Test with Belavezha script**

Run teleprompter export on `video-projects/_IN_PRODUCTION/30-belavezha-accords-2025/SCRIPT.md`:
- Verify all B-roll notes stripped
- Verify source citations stripped
- Verify clean readable output
- Check word count matches expected (~900 words)
</task>
