# Write Script Command

Generate a complete video script using the creator's natural voice and proven structure.

## Your Task

Create an 8-9 minute video script following the "both extremes are wrong" pattern with natural conversational voice.

## Required Information

**Topic:** $ARGUMENTS

Before you start, ask:
1. **What's the specific myth you're debunking?**
2. **What modern event/hook connects to this? (2024-2025)**
3. **What are the two extreme narratives?**
   - Extreme A (usually dismissive):
   - Extreme B (usually oversimplified):
4. **What research have you done?** (sources, key findings)

---

## Process

### Step 1: Read Required Files
- **VOICE-GUIDE.md** - Natural speech patterns
- **CLAUDE.md** - Channel mission
- **fact-checking-protocol.md** - Source standards

### Step 2: Identify Structure
- What's Extreme A?
- What's Extreme B?
- What's the nuanced reality?
- What modern events connect?

### Step 3: Research & Gather Evidence
- Primary sources
- Smoking gun quotes
- Modern examples (2024-2025)
- Pre-existing research provided

### Step 4: Draft Script
Follow this structure:

**OPENING (0:00-1:00):**
- Modern hook
- Frame BOTH extremes
- "Both are wrong"
- Promise to check documents
- Stakes

**EVIDENCE (1:00-7:00):**
- Visual payoff first
- Layer evidence incrementally
- **Modern connection every 90 seconds**
- Use "the biggest deal is..."
- Stack 3-4 examples
- Include smoking gun quotes
- Max 4 dates per section

**SYNTHESIS (7:00-8:30):**
- "Let me bring this back..."
- Restate Extreme A + why dangerous
- Restate Extreme B + why dangerous
- Nuanced reality
- Stakes: "people are dying"

**CTA (8:30-9:00):**
- Sources available
- Subscribe

### Step 5: Voice Check
- [ ] Natural fillers throughout
- [ ] "The biggest deal is..." used
- [ ] "Both extremes" pattern complete
- [ ] Self-corrections included
- [ ] Conversational source citations
- [ ] No formal language

### Step 6: Structure Check
- [ ] Modern hooks every 90 seconds
- [ ] No date overload
- [ ] Visual breaks every 30-45 sec
- [ ] Length: 850-1000 words

### Step 7: Read Aloud Test
Does every sentence sound like something the creator would actually say?

---

## Output Format

Provide the script with:

**Header:**
```
# [Title]

**Target Length:** 8-9 minutes (~XXX words)
**Modern Hook:** [Brief description]
**Both Extremes:**
- Extreme A: [Description]
- Extreme B: [Description]
**Nuanced Reality:** [One sentence]
```

**Script sections:**
- Mark timestamps
- Mark visual breaks
- Include [PAUSE] markers for emphasis
- Note modern connections with **[MODERN HOOK]**

**Footer:**
```
## SCRIPT CHECKLIST
- [x] Both extremes framed in opening
- [x] Modern relevance every 90 seconds
- [x] No date overload (max 4 per section)
- [x] Natural voice throughout
- [x] Visual breaks marked
- [x] Synthesis closes loop
- [x] Length: XXX words (X:XX minutes)
```

---

## Key Reminders

- **Sound conversational, not formal**
- **Frame both extremes in opening - non-negotiable**
- **Modern connection every 90 seconds - measure this**
- **One layered argument, not 3 claims**
- **Read it aloud - would the creator say this?**

Start by confirming you have:
1. The topic
2. The myth
3. Both extreme narratives
4. Modern hook
5. Available research

Then proceed with the script.
