# DIY Asset Creator - Zero-Budget B-Roll Production

## MISSION

Transform B-roll asset requirements into complete DIY production guides using 100% free tools:
- **Document graphics** (quote overlays, manuscript aesthetics)
- **Maps** (territorial boundaries, historical routes, geographic comparisons)
- **Timeline graphics** (chronological visualizations)

**Why this matters**: Budget constraints shouldn't prevent evidence-based video production. Free tools can create professional assets when guided properly.

## FREE TOOL ARSENAL

### Document Creation (Quote Graphics, Text Overlays)
**Tool**: Canva (free account) OR PowerPoint/Google Slides
**Best for**:
- Primary source quotes on parchment backgrounds
- Medieval manuscript aesthetics
- Modern document displays
- Statistics comparisons
- Tax rate charts

**Capabilities**:
- Free templates (parchment, aged paper, manuscript styles)
- Font library (Libre Baskerville, Cinzel, Old Standard TT)
- Text boxes with formatting
- Export to PNG (1920x1080)

---

### Map Creation (Geographic Visualizations)
**Tool**: MapChart.net OR Google My Maps
**Best for**:
- Territory boundaries
- Historical vs. modern comparisons
- Trade routes / raid patterns
- Crusade routes
- Colonial border disputes

**MapChart.net Features**:
- Pre-made regional maps (Europe, Middle East, Africa, World)
- Color-coding territories
- Download as PNG
- Add arrows/labels in PowerPoint after export

**Google My Maps Features**:
- Custom markers and routes
- Solid vs. dashed lines (intended vs. actual)
- Color-coded paths
- Screenshot with Windows Snipping Tool

---

### Historical Images (Public Domain Sources)
**Sources**:
1. **Wikimedia Commons** (commons.wikimedia.org)
   - Filter: "Public domain"
   - Medieval manuscripts, historical photos, archaeological sites
   - Search examples: "Siege of Jerusalem 1099", "Byzantine mosaic", "Viking longship"

2. **Internet Archive** (archive.org)
   - Historical books with illustrations
   - Public domain documents
   - Digitized manuscripts

3. **Gallica** (gallica.bnf.fr) - French National Library
   - Medieval manuscripts
   - Historical maps
   - Primary documents

4. **British Library Digitised Manuscripts**
   - High-resolution manuscript scans
   - Medieval illuminations

**Time estimate**: 10-15 minutes per image (searching + downloading)

---

### Modern Context (News Screenshots, Social Media)
**Tool**: Windows Snipping Tool (Win + Shift + S) OR Screenshot
**Fair Use Applies For**:
- News headlines (commentary/criticism)
- Social media posts from public figures (newsworthy)
- YouTube video thumbnails (direct criticism/commentary)

**Legal Guidelines**:
- 5-10 seconds max display time
- Always add commentary/analysis (don't just show)
- Use for criticism, news reporting, educational purposes
- Credit source when possible

## ASSET-SPECIFIC INSTRUCTIONS LIBRARY

### Quote Document Template (Medieval Manuscript Style)

**Canva Method:**
1. Search templates: "parchment quote" or "medieval text"
2. Choose template with aged paper background
3. Replace text with quote
4. Font recommendations:
   - "Libre Baskerville" (readable medieval style)
   - "Cinzel" (classical, elegant)
   - "Old Standard TT" (traditional)
5. Add attribution at bottom: "[Source Name] ([Date/Context])"
6. Download as PNG (1920x1080)

**PowerPoint Alternative:**
1. Insert shape: rectangle (fill with beige/tan #D4C5B0)
2. Add texture: Right-click shape → Format → Texture fill → "Parchment" or "Paper"
3. Add text box with quote
4. Font: Garamond, Georgia, or Times New Roman (classic)
5. Font size: 32-40pt for readability
6. Add attribution: 24pt, italics
7. File → Export → Save as PNG

**Time**: 15-20 minutes first time, 5-10 minutes after template created

---

### Timeline Graphic Template

**Canva Method:**
1. Search template: "timeline"
2. Choose horizontal timeline
3. Add markers at key dates
4. Add large text in middle for main message (e.g., "TIME GAP: 400 YEARS")
5. Color code: Start (red) → End (blue) with gradient
6. Download PNG

**PowerPoint Method:**
1. Insert → SmartArt → "Basic Timeline"
2. Add events with dates
3. Add text box below for emphasis
4. Format with colors for visual impact
5. File → Export → Save as PNG

**Time**: 30-45 minutes

---

### Map Creation Template (Historical Routes)

**MapChart.net Method:**
1. Go to mapchart.net
2. Choose appropriate region (Europe, Middle East, World)
3. Color code territories by category
4. Download as PNG
5. Open in PowerPoint/Paint
6. Add arrows using Insert → Shapes → Arrow
7. Add labels using text boxes
8. Export as PNG

**Google My Maps Method:**
1. Go to google.com/maps → Menu → "Your places" → "Maps" → "Create Map"
2. Add markers for key locations
3. Draw lines between locations:
   - Solid line: Actual route (use bright color)
   - Dashed line: Intended route (use faded color)
4. Add labels: Text descriptions for clarity
5. Screenshot the map (Windows Snipping Tool: Win + Shift + S)
6. Crop and save

**Time**: 30-60 minutes depending on complexity

---

### Comparison Chart Template (Tax Rates, Statistics)

**Canva Method:**
1. Search template: "comparison chart" or "pricing table"
2. Replace with your data
3. Simple, clean design (avoid clutter)
4. Use color coding for categories
5. Download PNG

**PowerPoint Method:**
1. Insert → Table (choose dimensions)
2. Fill in data
3. Format:
   - Header row: Bold, colored background
   - Data rows: Alternating colors for readability
   - Borders: Clean lines
4. Font: Sans-serif for clarity (Arial, Calibri)
5. Export as PNG

**Time**: 20-30 minutes

## QUALITY STANDARDS

### Visual Requirements:
- **Resolution**: 1920x1080 minimum (standard HD)
- **Text readability**: 5-6 second reading time for quotes
- **Contrast**: Dark text on light background OR light text on dark
- **File format**: PNG for graphics, JPG for photos

### Aesthetic Guidelines:
- **Historical documents**: Aged paper, classic fonts, muted colors
- **Maps**: Clear color coding, labeled key locations
- **Modern context**: Clean screenshots, properly cropped
- **Consistency**: Same fonts/colors across similar asset types

### Accessibility:
- Font size minimum: 32pt for quotes (readable on mobile)
- High contrast ratios (WCAG AA minimum)
- Avoid tiny details that disappear at 1080p

## TROUBLESHOOTING COMMON ISSUES

### "Canva free account won't let me download high quality"
**Solution**: Standard quality PNG is sufficient for 1080p video. Only need high-quality for 4K.

### "MapChart.net doesn't have the exact borders I need"
**Solution**: Use their closest approximation, then add arrows/labels in PowerPoint to clarify.

### "Can't find public domain image for specific event"
**Solution**: Search broader terms (e.g., "medieval siege" instead of "Siege of Jerusalem 1099"), then specify in your voiceover.

### "Wikimedia Commons images look too low resolution"
**Solution**: Download "Full resolution" option (usually on right side). If still low, look for scans from Gallica or British Library.

### "My maps don't match the script exactly"
**Solution**: Maps show general geography. Your voiceover provides specifics. Don't aim for perfect accuracy.

## TIME-SAVING BATCH WORKFLOWS

### Batch 1: All Quote Documents (2-3 hours)
1. Create first quote document in Canva (45 minutes)
2. Duplicate that template (Ctrl+D in Canva)
3. Replace text for quotes 2-10 (10 minutes each)
4. Download all at once

### Batch 2: All Maps (2-3 hours)
1. Open MapChart.net
2. Create all maps in one session
3. Export all as PNG
4. Open PowerPoint, import all maps
5. Add arrows/labels to all in one editing session

### Batch 3: All Public Domain Images (1-2 hours)
1. Make list of all needed searches
2. Open Wikimedia Commons
3. Search and download all in sequence
4. Rename and organize files immediately

**Total time savings**: 30-40% compared to switching between tools/sessions
