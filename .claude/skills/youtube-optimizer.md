# YouTube Metadata Optimizer Skill

## Description
Generates optimized YouTube titles, descriptions, tags, and chapters for History vs Hype videos. Takes video summary/transcript and creates complete publishing package following channel best practices.

## When to Use
- After filming is complete
- When you have a transcript or detailed video summary
- Before uploading to YouTube
- When you need SEO optimization

## What This Skill Does
1. Analyzes your video content
2. Generates 3 optimized title options (45-60 chars)
3. Creates complete description with:
   - Hook leading with strongest evidence
   - Timestamps/chapters
   - Key findings bullets
   - Primary sources listed
   - Modern relevance section
   - Call to action
4. Generates comprehensive tag list
5. Creates YouTube chapters (optimal number, best practices)
6. Provides thumbnail recommendations
7. Suggests upload timing
8. Includes pre-publish checklist

## Inputs Needed
- Video length
- Video topic/summary
- Main claims being fact-checked
- Key evidence/findings
- Primary sources used
- Target keywords (optional)

## Output Format
Complete copy-paste ready package:
- Title options
- Full description
- Tags
- Chapters
- Thumbnail specs
- Upload strategy
- Community post template

## Channel-Specific Optimization
- Uses History vs Hype voice (academic, evidence-based)
- Emphasizes primary sources
- Highlights modern relevance
- Follows documentary style
- Optimized for males 25-44 demographic
- Keywords for political/history content

## Best Practices Applied
- Title: 45-60 characters, includes topic + differentiator
- Description: Strongest evidence first, all sources listed
- Chapters: 5-8 max, logical breaks, descriptive
- Tags: Primary keywords, long-tail variations
- SEO: Optimized for search traffic

## Example Usage
**You say:** "I need YouTube metadata for my 15-minute video fact-checking Reagan's economic claims. I show primary sources from OMB historical tables proving trickle-down didn't work."

**Skill generates:**
- 3 title options
- Complete description with sources
- 20+ relevant tags
- 6 YouTube chapters
- Thumbnail recommendation
- Full publishing package
