## WHEN TO USE NOTEBOOKLM FOR SCRIPTS

### Use Case 1: Argument Structure Check (Before Writing)

**Upload:** Your preliminary research + Toulmin/Seixas books

**Prompt:**
```
I'm writing a script debunking the myth that [MYTH]. My preliminary evidence is:

1. [Evidence point 1]
2. [Evidence point 2]
3. [Evidence point 3]

Using Toulmin's model of argumentation:
- What is my core CLAIM?
- What EVIDENCE supports it?
- What is the WARRANT (logical bridge)?
- What BACKING supports my warrant?
- Are there any unstated assumptions I need to make explicit?

Then apply Seixas's Big Six:
- Which of the six concepts (Significance, Evidence, Continuity/Change, Cause/Consequence, Perspective, Ethics) should I emphasize?
- How can I structure the argument to TEACH historical thinking, not just correct facts?
```

**Output:** Clean argument skeleton before you start writing

### Use Case 2: Identity Stake Assessment (Pre-Production)

**Upload:** Kahan + Klein sources

**Prompt:**

```
I'm creating a video about [TOPIC - e.g., "The 1961 Somaliland referendum"].

Based on identity-protective cognition research:

3. What psychological needs does the dominant myth fulfill? (belonging, status, coherence)
4. How should I structure the opening to reduce identity threat?
5. What self-affirmation strategies should I use?

Provide specific language examples based on the research.
```

**Output:** Identity map + opening strategy

### Use Case 3: Backfire Effect Prevention (During Writing)

**Upload:** Lewandowsky Debunking Handbook + your script draft

**Prompt:**
```
Review this script section for potential backfire effects:

[PASTE SCRIPT SECTION]

Check for:
1. **Familiarity Backfire:** Am I repeating the myth too much? Does the myth appear in headlines/section titles?
2. **Overkill Backfire:** Am I providing too many counter-arguments? Should I cut to 3 key points?
3. **Worldview Backfire:** For identity-protective viewers, will this correction strengthen their belief?

Suggest revisions using:
- Fact-first headlines (lead with correct fact, not myth)
- Alternative explanations (fill mental model gaps)
- KISS principle (3 key points max)
- Self-affirmation language if needed
```

**Output:** Revised script with backfire risks minimized

### Use Case 4: Historical Thinking Integration (Quality Gate)

**Upload:** Wineburg + Seixas + your draft script

**Prompt:**
```
Analyze this script draft:

[PASTE FULL SCRIPT]

Using Wineburg's framework:
1. Am I committing PRESENTISM? (Judging past by present standards)
2. Am I using SOURCING properly? (Who created document, when, why)
3. Am I showing CONTEXTUALIZATION? (Climate of opinion at the time)

Using Seixas's Big Six:
1. **Significance:** Did I explain WHY this matters (deep consequences for many over time)?
2. **Evidence:** Did I show sourcing AND corroboration (2+ sources for key claims)?
3. **Continuity/Change:** Did I show what stayed same vs. what changed?
4. **Cause/Consequence:** Did I move beyond "great man" to structural factors?
5. **Perspective:** Did I treat the past as "foreign country" with different norms?
6. **Ethics:** Did I make reasoned moral judgments while respecting context?

For each Big Six concept, either:
- ✅ Confirm I did it well
- ⚠️ Suggest specific improvements with example language
- ❌ Flag if completely missing
```

**Output:** Big Six audit with specific fixes

### Use Case 5: Trust-Building Language (Final Polish)

**Upload:** Besley science communication + your script

**Prompt:**
```
Review this script for trustworthiness signals:

[PASTE SCRIPT]

Using Besley's framework, rate 1-10 and suggest improvements for:

1. **Warmth (Caring):** Do I show I care about getting history right?
   - Suggest warmer language that doesn't sacrifice authority

2. **Integrity (Honesty):** Am I transparent about sources and limitations?
   - Flag any claims that need source attribution
   - Suggest language for admitting uncertainty

3. **Willingness to Listen:** Do I acknowledge valid points from the "other side"?
   - Identify places to steelman opposing view
   - Suggest "procedural fairness" language

4. **Competence:** Do I demonstrate expertise without arrogance?
   - Check balance of confidence vs. humility

5. **Shared Values:** Do I affirm values the audience holds?
   - For high-identity sections, suggest self-affirmation language

Provide 3-5 specific revisions with before/after examples.
```

**Output:** Trust-optimized language revisions

### Use Case 6: Alternative Explanation Development (Gap Filling)

**Upload:** Your research + Lewandowsky

**Prompt:**
```
I'm debunking the myth that [MYTH].

The problem: Simply saying "that's wrong" leaves a mental gap. People prefer incorrect models to no model.

Based on my research, develop a plausible alternative explanation that:
1. Accounts for all observed features of the historical event
2. Provides clear causation (A → B → C)
3. Fills the "why" gap the myth was filling
4. Is evidence-based (cite specific sources from my research)

Structure the alternative as:
- What ACTUALLY happened
- WHY it happened (causes)
- WHAT consequences resulted
- WHY the myth is appealing but inaccurate

Provide the alternative in script-ready language (conversational, flowing, under 200 words).
```

**Output:** Complete alternative explanation ready to insert

### Use Case 7: Source Credibility Audit (Inoculation)

**Upload:** Your research sources + Lewandowsky (source suspicion)

**Prompt:**
```
I'm debunking [MYTH]. The myth originated with [SOURCE/PERSON].

Using "source credibility suspicion" technique:
1. Who created/propagated this myth originally?
2. What were their vested interests? (political, religious, economic, cultural)
3. When and why was the myth useful to them?
4. How did it spread from [creator] to [mainstream acceptance]?
5. What institutional backing gave it legitimacy? (textbooks, media, political endorsement)

Provide a 150-word script section that:
- Exposes the chain of propagation
- Explains vested interests without conspiracy theorizing
- Shows why the myth was "useful fiction"
- Uses calm, forensic tone (no outrage)

Include specific names, dates, and motivations based on sources.
```

**Output:** Source credibility section ready to insert

### Use Case 8: Sedimented Histories Approach (Contested Topics)

**Upload:** Your research + Seixas/Cauvin

**Prompt:**
```
I'm covering [CONTESTED HISTORICAL EVENT - e.g., territorial dispute].

There are competing national narratives:
- Narrative A (Country X): [Describe]
- Narrative B (Country Y): [Describe]

Instead of picking a side, I want to use "sedimented histories" approach:
- Present both narratives as historical artifacts
- Explain why each group tells this story (identity function)
- Show which claims are evidence-based vs. politically useful
- Teach viewers HOW to assess narrative plausibility

Using Seixas's "narrative plausibility" criteria:
1. **Evidentiary plausibility:** Which narrative is better supported by documents?
2. **Causal plausibility:** Which provides convincing link between cause and effect?
3. **Empathetic plausibility:** Which accounts for perspectives of all actors?

Provide:
- Opening framing that sets up "rival histories"
- Side-by-side comparison structure
- Plausibility assessment that's fair but evidence-based
- Closing that acknowledges legitimate grievances on both sides while being clear about factual record

Script-ready language, 400-500 words.
```

**Output:** Full contested history section using historiographic gaze

### Use Case 9: Gateway Effect Warning (Modern Relevance)

**Upload:** Conspiracy belief research + your topic

**Prompt:**

```
Research shows a "gateway effect"—belief in one historical myth makes it easier to believe contemporary conspiracies.

Based on the sources:
1. How does THIS specific myth connect to modern conspiracy thinking?
2. What contemporary movements/beliefs cite this myth as "proof"?
3. What is the psychological mechanism linking historical myth → current distrust?

Provide a 100-150 word script section that:
- Connects this historical myth to current conspiracy beliefs
- Cites specific research on gateway effects
- Uses calm, factual tone (no moral panic)
- Frames as "lesson about how misinformation persists"

Include any relevant statistics from research (trust in science, conspiracy belief prevalence, etc.).
```

**Output:** Modern relevance section with research-backed connections

### Use Case 11: International Comparison Discovery (NEW 2025-01-12)

**Purpose:** Find "Unlike [Country]..." comparisons that strengthen your argument

**Upload:** Your research sources + any relevant comparative history

**Prompt:**
```
I'm covering [YOUR TOPIC] for a YouTube history video.

Based on the sources, identify international comparisons I can use:

1. **Direct Contrast:** Find cases where another country faced the same situation but handled it differently
   - Example: "Unlike Britain, which compensated slaveholders, France made the enslaved pay"
   - What countries provide useful contrasts to [subject]?

2. **Parallel Cases:** Find similar events in different countries that illustrate the same principle
   - Example: Louisiana Purchase price vs. Haiti indemnity
   - What parallel cases strengthen my argument?

3. **Opposite Outcomes:** Find countries with similar starting conditions but different results
   - Example: Nogales, Arizona vs. Nogales, Mexico
   - What "natural experiments" exist?

For each comparison found, provide:
- Country/case name
- Key comparison point
- Specific numbers/dates if available
- Script-ready sentence using "Unlike [Country]..." pattern

Prioritize comparisons with:
- Dramatic contrast (viewers immediately grasp the point)
- Specific numbers (not vague comparisons)
- Familiar reference points (countries viewers know)
```

**Output:** 3-5 international comparisons with script-ready language

### Use Case 12: Data Comparison Hook Development (NEW 2025-01-12)

**Purpose:** Find specific data points for Kraut-style opening hooks

**Upload:** Your research sources + economic/demographic data

**Prompt:**
```
I need data for a comparison-style opening hook (like Kraut's Nogales video).

My topic is [YOUR TOPIC].

Find two data points I can contrast that:
1. Are dramatically different but comparable
2. Have specific numbers (not vague descriptions)
3. Illustrate my thesis before I state it
4. Come from the same category (income vs. income, territory vs. territory, time vs. time)

Categories to search:
- Economic data (GDP, income, debt amounts, trade values)
- Geographic data (territory size, population, density)
- Temporal data (time periods, duration of events)
- Quantitative outcomes (deaths, displacement, migration)

Provide data in this format:

**Data Point A:** [Specific number with source]
**Data Point B:** [Specific number with source]
**The Contrast:** [Why this comparison is striking]
**Script-Ready Hook:**
"[Data A]. Just [comparison] away/later... [Data B]. Same [category]. Different [outcome]. What created this gap?"

Find at least 2 potential comparison hooks.
```

**Output:** Data comparison hooks ready to use in script opening

### Use Case 13: Competitor Differentiation Research (NEW 2025-01-12)

**Purpose:** Find angles that distinguish your video from competitors

**Upload:** Your research sources (competitors' transcripts if you have them)

**Prompt:**
```
I'm creating a video on [YOUR TOPIC].

I've identified that most videos on this topic cover:
- [Angle 1 that's overdone]
- [Angle 2 that's overdone]
- [Angle 3 that's overdone]

Based on my research sources, find UNIQUE angles I can cover that competitors miss:

1. **Primary Source Angle:** What documents can I SHOW ON SCREEN that competitors don't show?
   - Treaties, ordinances, legal texts, census data, correspondence

2. **Counterargument Angle:** What legitimate counter-evidence exists that competitors ignore?
   - Scholarly debates, alternative interpretations, nuance

3. **Mechanism Angle:** What "HOW" questions do competitors skip?
   - How was debt calculated? How did the system work?

4. **Comparison Angle:** What international comparison would set my video apart?
   - What "Unlike [Country]..." pattern is unused?

5. **Modern Relevance Angle:** What fresh modern connection exists?
   - Recent news, 2024-2025 developments, ongoing legal cases

For each unique angle, assess:
- Is this supported by my sources? [Yes/Needs more research]
- Would viewers find this surprising? [Yes/Somewhat/No]
- Does this support my thesis? [Yes/Tangential/No]

Provide 3-5 differentiation opportunities with script-ready framing.
```

**Output:** Unique angles competitors miss that you can cover

### Use Case 14: Timeline Extraction

**Purpose:** Converting scattered date references into chronological sequence for script structure

**Upload:** All primary and secondary sources for your topic

**Prompt:**
```

Using ALL uploaded sources, extract:
1. Every dated event mentioned (year, month/day if available)
2. The event description (1 sentence max)
3. The source citation (Title, page number)

Format as:
**[DATE]:** [Event description] [Citation number]

Sort from earliest to latest. If multiple sources mention same event, list all citations.

Focus on events central to [SPECIFIC ASPECT - e.g., "the myth's creation" or "territorial dispute escalation"].

Flag any date discrepancies between sources.
```

**Example Usage:**

>
> Focus on events central to the myth's creation and propagation.

**Output sample:**
> **1828:** Washington Irving publishes *A History of the Life and Voyages of Christopher Columbus*, inventing the Salamanca flat-earth debate [1]

```
1. [Claim 1 with number]
2. [Claim 2 with number]
3. [Claim 3 with number]

For EACH claim, search ALL uploaded sources and provide:

| Claim | Source 1 | Source 2 | Source 3 | Verdict |
|-------|----------|----------|----------|---------|
| [The claim] | [Value + citation] | [Value + citation] | [Value + citation] | VERIFIED / CONFLICTING / SINGLE SOURCE |

For CONFLICTING data:
- Explain the likely reason for discrepancy (different methodology, date, scope)
- Recommend which figure to use and why
- Suggest hedge language if uncertainty remains

For SINGLE SOURCE data:
- Flag reliability concern
- Suggest alternative phrasing that acknowledges limitation
```

**Example Usage:**

**Context:** Bir Tawil video - verifying territory sizes

**Customized prompt:**
> Verify these numbers:
> 1. Bir Tawil is 2,060 km²
> 2. Hala'ib Triangle is 20,580 km²
> 3. Manganese reserves exceed 700,000 tons

**Output sample:**
> | Claim | Source 1 | Source 2 | Verdict |

### Use Case 17: Counterargument Mapping

**Purpose:** Steelmanning opposing views with their best evidence (Alex O'Connor intellectual honesty)

**Upload:** All sources including those presenting opposing viewpoint

**Prompt:**
```
I'm arguing [YOUR THESIS] in my video about [TOPIC].

Before finalizing, I need to honestly assess the strongest counterarguments.

1. **Steelman the opposition:**
   - What is the BEST version of the argument against my thesis?
   - What evidence from my sources supports this counterargument?
   - What legitimate concerns does this position address?

2. **Evaluate the counterargument:**
   - Why does it ultimately fall short? (Evidence gaps, logical problems, cherry-picking)
   - What does it get RIGHT that I should acknowledge?
   - How can I address it without strawmanning?

3. **Script integration:**
   - At what point in my video should I address this?
   - Provide 50-100 word script section that:
     * Presents the counterargument fairly
     * Acknowledges valid points
     * Explains why the evidence points elsewhere
     * Uses "procedural fairness" language (Besley framework)

Format the counterargument section using Alex O'Connor's pattern:
"Now, someone might reasonably argue that... And they'd have a point, because... However, when we look at [evidence], we see..."
```

**Example Usage:**

**Context:** Dark Ages myth video - counterargument that "some decline did happen"

**Customized prompt:**
> I'm arguing that the "Dark Ages" myth is largely invented.
> Steelman the counterargument that genuine decline occurred.

**Output sample:**
> **Steelmanned argument:** Genuine decline DID occur in certain metrics - urban population dropped 90% in some regions, long-distance trade collapsed, literacy rates fell dramatically. Historians like Bryan Ward-Perkins document this material decline extensively.
>
> **What it gets right:** Material culture genuinely simplified. Roman infrastructure wasn't maintained. Measurable decline in pottery quality, building techniques, coinage.
>
> **Why it falls short:** "Decline" ≠ "Dark." The myth claims INTELLECTUAL darkness (flat earth belief, no learning), but monasteries preserved classical texts, Carolingian Renaissance showed cultural achievement, Islamic world flourished with trade connections.
>
> **Script section:**
> "Now, someone might reasonably point out that material conditions genuinely declined after Rome fell. And they'd have a point - Bryan Ward-Perkins documents how pottery quality, building techniques, and urban populations all dropped significantly. But here's what the 'Dark Ages' myth gets wrong: material simplification isn't the same as intellectual darkness..."

**Output:** Steelmanned counterargument with fair script section

## TIPS FOR EFFECTIVE PROMPTS

**Be specific about output format:**
- "Provide in script-ready language, 200 words"
- "Use conversational tone, contractions, flowing delivery"
- "Include specific page numbers from sources"

**Reference your channel voice:**
- "Use calm prosecutor tone, not outrage"
- "Confident about evidence, humble about interpretation"
- "Kraut-style causal chains, Alex O'Connor intellectual honesty"

**Iterate, don't accept first output:**
- If response is too academic, ask: "Rewrite in conversational voice for YouTube"
- If too vague, ask: "Provide specific examples with page numbers"
- Use Interactive Mode to refine

**Combine prompts strategically:**
- Run identity assessment BEFORE drafting opening
- Run backfire check AFTER writing but BEFORE filming
- Run Big Six audit as final quality gate
