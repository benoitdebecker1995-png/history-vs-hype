# Gap Database - Underserved Topics

**Purpose:** Identify underserved topics where History vs Hype can provide unique value.
**Use this for:** Topic selection, prioritizing video ideas, finding competitive advantages.

*Last updated: 2026-01-23*

---

## What Makes a Topic "Underserved"

A topic is underserved if ANY of these apply:
- **Low competition:** Few videos exist on the topic
- **Missing angle:** Videos exist but miss the primary-source / myth-busting approach
- **Poor quality:** Existing videos are shallow, inaccurate, or outdated (pre-2020)

---

## Scoring Criteria

| Factor | Score | Description |
|--------|-------|-------------|
| **Demand** | 1-5 | Search volume, comment requests, news relevance |
| **Competition** | 1-5 | 5 = no good coverage, 1 = saturated |
| **Channel Fit** | 1-5 | History-first, myth-busting, primary sources |
| **Modern Hook** | +1 bonus | Current event connection (2024-2026) |

**Total Score:** Demand + Competition + Fit + Hook = Max 16

**Priority Thresholds:**
- 13-16: High priority - strong opportunity
- 9-12: Medium priority - good opportunity
- 5-8: Low priority - consider if passion project
- <5: Skip - poor fit or saturated

## Gap Sources

Where to find new gaps:

1. **Competitor comments:** "You should cover X" on similar channels
2. **VidIQ keyword gaps:** Search terms with demand but low competition
3. **Academic papers:** Recent scholarship without YouTube coverage
4. **News events:** Current events needing historical context
5. **OUTLIER-TOPIC-IDEAS.md:** Validated topic patterns
