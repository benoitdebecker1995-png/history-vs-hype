# Closing & Synthesis Templates

**Use these templates for the final 60-90 seconds. Copy, paste, fill in the blanks.**

---

## TEMPLATE 1: Return to Both Extremes

**Best for:** Videos using "Both Extremes Wrong" format

```
So, where does that leave us?

Remember the two narratives we started with?

One side said [Extreme A]. [Brief recap of who believes this].

The other side said [Extreme B]. [Brief recap of who believes this].

[Beat]

The reality is more complicated—and more interesting.

[Extreme A is wrong because]: [Specific evidence that contradicts it].

[Extreme B is wrong because]: [Specific evidence that contradicts it].

The actual story is [nuanced reality]. And that matters because [modern consequence].
```

**Example (Haiti):**
```
So, where does that leave us?

Remember the two narratives we started with?

One side said France bankrupted Haiti. Everything is colonial extraction.

The other side said Haitians did it to themselves. Internal corruption explains the poverty.

The reality is more complicated—and more interesting.

Yes, France extracted $560 million in today's dollars. That's documented.

But the Duvaliers stole $1.65 billion—three times more.

The actual story is both extraction AND corruption, in sequence. And that matters because you can't fix a problem you don't understand.
```

## TEMPLATE 2: Universal Pattern (Kraut-Style)

**Best for:** Videos about systems, patterns, or mechanisms

```
If you see the pattern, you understand something bigger.

This isn't about [specific topic]. It isn't about [specific location].

It's about [universal principle].

[Concrete example of pattern elsewhere].

[Another example if relevant].

That story applies way beyond [location in video].

[Why understanding this pattern matters for viewer].
```

**Example (Iran):**
```
If you see the pattern, you understand something bigger.

This isn't about Iran. It isn't about Islam.

It's about what happens when legitimate grievances get captured by authoritarian structures.

The Constitutional Revolution of 1906 wanted accountability. Khomeini gave them divine authority.

The same pattern plays out in Hungary, Turkey, Venezuela—movements that started with real complaints, ended with one leader who can't be questioned.

That story applies way beyond Iran.

If you want to understand why revolutions keep producing new tyrants, you have to understand how grievance becomes control.
```

## TEMPLATE 3: Document Callback

**Best for:** Videos built around primary source evidence

```
We started with [the document/treaty/law].

[What it said or showed].

Now you've seen [what actually happened / the full context].

[Beat]

The document is still [relevant today because...].

[Specific modern consequence or ongoing dispute].

[Why understanding the document matters].
```

**Example (Belize-Guatemala):**
```
We started with the 1859 treaty.

One line: "Guatemala cedes Belize in exchange for a road."

Now you've seen that Britain never built the road.

The treaty is still the basis of Belize's existence as a nation.

Guatemala still claims the entire territory. The ICJ will rule by 2027.

That's why understanding what the treaty actually says—and what it doesn't—matters for 400,000 people.
```

## TEMPLATE 4: Stakes Reminder

**Best for:** Videos about ongoing conflicts or active disputes

```
This isn't ancient history.

[Specific modern consequence happening now].

[Number of people affected / money at stake / lives impacted].

[Beat]

The myths we covered today—[Myth A], [Myth B]—they're still shaping [policy / debate / conflict].

[Who benefits from the myths continuing].

[Why accurate history matters for this specific issue].

That's why I do these videos. Primary sources. Full context. No shortcuts.

If you want more, please subscribe.
```

---

## TEMPLATE 5: Intellectual Humility Close (Alex O'Connor Style)

**Best for:** Videos on contested topics where you acknowledge uncertainty

```
Now, I want to be clear about something.

I'm not saying [Extreme A people] are stupid. I'm not saying [Extreme B people] are liars.

[Beat]

The history is complicated. Reasonable people can interpret [specific contested point] differently.

What I AM saying is: if you're going to have an opinion on [topic], you should know [key facts established in video].

[Specific fact 1]. [Specific fact 2]. [Specific fact 3].

Those are the facts. What you conclude from them is up to you.

But at least now you're working from the same evidence the historians use.
```

## TEMPLATE 6: Call to Action Integration

**Best for:** Any closing—integrate CTA naturally

```
[Synthesis content from templates above]

---

If you found this useful, please subscribe. It helps the channel grow.

Full sources are in the description—including [specific notable source].

[Optional: tease next video if relevant]

Thanks for watching.
```

**Natural CTA phrases:**
- "If you found this useful, please subscribe."
- "Full sources in the description."
- "Thanks for watching."

**Avoid:**
- "Smash that subscribe button"
- "Drop your thoughts below"
- "Don't forget to like"

---

## CLOSING CHECKLIST

Before finalizing your closing:

- [ ] Returns to opening premise (both extremes, document, question)
- [ ] Answers the "so what?" question explicitly
- [ ] Modern relevance restated or implied
- [ ] NO forbidden phrases
- [ ] CTA is polite and natural
- [ ] Ends with concrete takeaway, not vague sentiment
- [ ] Under 90 seconds when read aloud
