# Scriptwriting Quick Reference Card

**One-page reference for all scriptwriting rules. Print this.**

---

## FORBIDDEN PHRASES (Never Use)

```
❌ "Let me show you something"
❌ "Here's where it gets interesting"
❌ "And that's the key insight"
❌ "Buckle up"
❌ "Stay with me here"
❌ "Here's the thing"
❌ "You won't believe"
❌ "SHOCKING"
❌ "Drop your thoughts below"
❌ "Smash that subscribe"
```

---

## APPROVED PHRASES (Copy-Paste Ready)

### Introducing Surprising Facts
```
This might surprise you but...
The reality is...
Here's what the sources actually show:
```

### Transitions Between Sections
```
On top of that...
Furthermore...
That being said...
And that brings us to...
Which created a problem:
That's the context for what happened next.
```

### Showing Documents
```
As you can see in this document...
Here's what [the treaty/letter] actually says:
[Name]'s exact words:
```

### Acknowledging Other Side
```
We have to keep in mind...
If we are being fair...
From their point of view...
Now, I want to be clear...
There's a grain of truth here.
```

### Making Points Land
```
This is crucial.
The reality is...
To summarize...
In conclusion...
If you take away only one thing...
```

### CTAs
```
Please subscribe if you want more.
Full sources in the description.
```

## TERM INTRODUCTION PATTERN

**Every name/term needs context on first use:**

```
[Term] — [plain language definition in same breath]
```

**Examples:**
```
The Qajars — a dynasty that had ruled Iran since 1789
The Great Game — a rivalry between Britain and Russia for Central Asia
Velayat-e Faqih — "guardianship of the jurist," the idea that one Islamic scholar should hold ultimate political power
The Basij — volunteer militias loyal to the Supreme Leader
```

---

## QUOTE INTEGRATION (3-Step)

**1. SETUP** (claim in your own words)
**2. QUOTE** (evidence)
**3. IMPLICATION** (why it matters)

**Example:**
```
In 2016, the Guardian Council disqualified 99 percent of reformist candidates.  ← SETUP

Parsa documents exactly what happened:
> "The council eliminated 99 percent of 3,000 reformists."  ← QUOTE

Ninety-nine percent. Before a single vote was cast.  ← IMPLICATION
```

**DON'T:**
```
> "The council eliminated 99 percent..."
That's from Parsa's research.  ← Quote dropped without setup
```

## TRANSITION BRIDGES

**Every section change needs a "why are we talking about this now" sentence:**

| From → To | Bridge Example |
|-----------|----------------|
| Character → Issue | "Mossadegh's conviction would be tested on one issue: oil." |
| Event → Consequence | "And that's when something unprecedented happened." |
| Problem → Solution | "So they found another way." |
| Section → New Section | "That's the context for what happened next." |
| Evidence → Implication | "Do you see what this means?" |

---

## IMPLICATION TRIGGERS

**After every major fact, answer "why does this matter?"**

```
That's how [adjective] X was.
Which meant...
In other words...
The implication:
Even [surprising detail]. That tells you something.
```

## OPENING PATTERNS

### Document Hook
```
This document [did X]. [Date]. [Key detail].
[Zoom to specific text]
[What it shows]
```

### Both Extremes
```
If you ask online about [topic], you'll get two answers.
One side says [X].
The other side says [Y].
Both are wrong. Here's why.
```

### Surprising Fact
```
This might surprise you, but [unexpected fact].
That's [timeframe]. [Context that makes it surprising].
```

---

## SECTION OPENINGS

**Start with concrete details, not abstractions:**

```
❌ "The situation in Iran was complicated by foreign interference."
✅ "December 1905. Tehran. The governor orders merchants beaten."

❌ "Let's examine the economic factors."
✅ "By 1914, 80 percent of Haiti's budget went to France."
```

---

## UNIVERSAL ENDINGS

**Connect specific topic to broader meaning:**

```
If you see the pattern, you understand something bigger:
This isn't about [specific]. It isn't about [specific].
It's about [universal principle].
That story applies way beyond [location].
```

## PRE-SCRIPT CHECKLIST

- [ ] Listed all names/terms that need introduction
- [ ] Defined each in plain language
- [ ] Identified section transitions — wrote bridge sentences
- [ ] For each quote: setup? implication?

## POST-SCRIPT SCAN

- [ ] Any term used before introduced?
- [ ] Any random transitions without bridges?
- [ ] Any quotes dropped without setup?
- [ ] Any facts without implications?
- [ ] Any forbidden phrases? (grep for them)
- [ ] Does it sound like me?


If any matches → rewrite using approved alternatives above.
