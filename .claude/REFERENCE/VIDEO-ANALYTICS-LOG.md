## How to Update This File

**After each video publishes:**
1. Wait 7 days for data to stabilize
2. Check YouTube Studio → Analytics → Retention
3. Screenshot retention graph
4. Log drop-off points with timestamps
5. Hypothesize causes
6. Add lessons learned
7. Update baseline metrics if new benchmark

**Monthly review:**
- Look for patterns across 3+ videos
- Compare performance by video type
- Update benchmarks if exceeded
