## Discovery Workflow

**Monthly check (Claude's discretion on exact timing):**

1. **VidIQ Scan:** Check "Channels Like Yours" or niche recommendations
2. **Outlier Hunt:** Search topics -> filter by views vs subscriber ratio
3. **Comment Mining:** Note channels mentioned positively in competitor comments
4. **Evaluate:** Does this channel have techniques we can learn from?

**When adding a channel:**
1. Note discovery signal (which metric stood out)
2. Watch 2-3 videos to understand their approach
3. Extract specific technique if valuable
4. Add to appropriate tier

**Non-history channels:** Include if techniques are applicable (editing, data presentation, pacing)

## Technique Extraction Process

When you find a valuable technique:

1. **Watch the video:** Note timestamp of technique in action
2. **Transcribe excerpt:** Get actual words used (transcript excerpts preferred)
3. **Categorize:** Which script phase? (opening, argument, evidence, closing)
4. **Add to library:** Update `PROVEN-TECHNIQUES-LIBRARY.md`
5. **Credit source:** Note creator and video in the entry
