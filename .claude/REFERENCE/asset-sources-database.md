## 🗺️ MAPS

### Historical Maps

| Source | URL | Best For | License |
|--------|-----|----------|---------|
| **Gallica (French National Library)** | https://gallica.bnf.fr | Ottoman, Colonial Africa, French colonies | Public Domain |
| **David Rumsey Map Collection** | https://www.davidrumsey.com | High-res historical maps, all regions | Fair use/educational |
| **Old Maps Online** | https://www.oldmapsonline.org | Searchable by location + date | Varies |
| **Library of Congress Maps** | https://www.loc.gov/maps | American history, colonial era | Public Domain |
| **British Library Maps** | https://www.bl.uk/maps | British Empire, colonial borders | Some restrictions |

**Download Steps (Gallica):**
1. Search for region + date (e.g., "Ottoman 1890")
2. Click image → Download icon
3. Select "Image" → "Full image"
4. Save as `[Region]_[Year]_Map.jpg`

### Modern Maps

| Source | URL | Best For | License |
|--------|-----|----------|---------|
| **MapChart.net** | https://mapchart.net | Colored country maps, borders | Free for non-commercial |
| **Wikimedia Commons Maps** | https://commons.wikimedia.org/wiki/Category:Maps | SVG vectors, clean borders | CC-BY or Public Domain |
| **Google My Maps** | https://www.google.com/maps/d | Custom routes, marked locations | Screenshot only |
| **Natural Earth** | https://www.naturalearthdata.com | GIS-quality base maps | Public Domain |

**Quick MapChart Workflow:**
1. Go to mapchart.net → Select map type
2. Click countries to color them
3. Add labels if needed
4. Download PNG → Save as `[Topic]_Modern_Map.png`

### Map Comparison Graphics

**Tool:** Canva (free)
**Template:** Search "Split comparison"
**Dimensions:** 1920 x 1080px

**Steps:**
1. Create design → 1920x1080
2. Add two rectangles (50% width each)
3. Insert historical map (left) + modern map (right)
4. Add text overlays: "1916" vs "Modern"
5. Highlight key differences (circles, arrows)
6. Export PNG

| **Avalon Project (Yale Law)** | https://avalon.law.yale.edu | Historical treaties, declarations |
| **Internet Archive** | https://archive.org | Correspondence, old documents |
| **UK National Archives** | https://www.nationalarchives.gov.uk | British colonial records |
| **JSTOR** | https://www.jstor.org | Academic papers with document excerpts |

**Document Screenshot Standard:**
- Resolution: 1920px width minimum
- Highlight key text with yellow overlay
- Add translation below if foreign language

### Court Rulings (ICJ)

**Direct source:** https://www.icj-cij.org/decisions

**Steps to find ruling:**
1. Go to ICJ website → "Decisions"
2. Search case name (e.g., "Nicaragua v Colombia")
3. Download judgment PDF
4. Screenshot specific paragraphs
5. Add case citation overlay

**Standard citation format:**
> [Case Name] ([Year]), ICJ Reports [Year], para. [X]

## 📊 GRAPHICS & INFOGRAPHICS

### Timeline Graphics

**Tool:** Canva (free)
**Search:** "Timeline infographic template"
**Customize:** Dates, colors, icons

**Standard History vs Hype timeline:**
- Dark background (black or dark gray)
- White/yellow text
- Color-coded events
- 1920 x 1080px

### Quote Cards

**Tool:** Canva
**Template:** Search "Quote" → Select dark/professional style

**Standard format:**
```
┌────────────────────────────────┐
│                                │
│   "[Quote text here]"          │
│                                │
│   — Source Name, Date          │
│     Document Title             │
│                                │
└────────────────────────────────┘
```

**Font:** Serif for quotes (Playfair, Georgia)
**Background:** Dark solid or subtle texture

### Data Visualizations

**Tool:** Canva or Google Sheets → Chart → Screenshot

**For statistics:**
1. Create chart in Google Sheets
2. Customize colors to match video style
3. Screenshot at 2x zoom
4. Import to video

- Use only brief clips (10-30 sec max)
- Add commentary/analysis over clip
- Transform meaning (fact-check context)
- Don't use as filler/decoration

**Public Domain Sources:**
- US government films (archive.org)

## 💡 TIPS BY ASSET TYPE

### When You Need a Map
1. Check Wikimedia Commons first (fastest)
2. If not found → David Rumsey or Gallica
3. If custom needed → MapChart.net
4. For comparison → Canva split template

### When You Need a Document
1. Check Avalon Project (treaties)
2. Check Internet Archive (correspondence)
3. Check specific archives (UK National, etc.)
4. Screenshot + highlight key text

### When You Need a Quote Card
1. Open Canva → Search "Quote"
2. Select dark/professional template
3. Change text + source
4. Export PNG
5. **Time:** 2-3 minutes per card

### When You Need a Timeline
1. Open Canva → Search "Timeline infographic"
2. Select horizontal style
3. Change dates + events
4. Color code by category
5. Export PNG
6. **Time:** 10-15 minutes
