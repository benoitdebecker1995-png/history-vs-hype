**Position:** After Phase 8.5 (Author Voice Lock) → During/After filming
**Type:** Non-writing execution guidance
**Purpose:** Translate approved script into mechanical filming and editing instructions

## VISUAL PRIORITY TIERS

### Tier 1: MUST-DO (Script fails without these)

| Category | Criteria |
|----------|----------|
| **Smoking gun documents** | Any document quoted with "reading directly from" |
| **Key payoff visual** | The single most important evidence moment |
| **Opening hook visual** | First map/image that establishes stakes |
| **Primary source readouts** | All `[PRIMARY SOURCE READOUT]` markers |

**Rule:** If Tier 1 asset is missing, do not film until resolved.

### Tier 2: IMPORTANT (Noticeably weaker without)

| Category | Criteria |
|----------|----------|
| **Supporting documents** | Cited but not quoted verbatim |
| **Timeline graphics** | Date sequences mentioned in script |
| **Secondary maps** | Geographic context beyond opening |
| **News footage** | Modern hook visuals |

**Rule:** Proceed with filming if missing, but prioritize before final edit.

### Tier 3: NICE-TO-HAVE (Polish, not essential)

| Category | Criteria |
|----------|----------|
| **Historical photos** | Atmosphere/context only |
| **B-roll variety** | Alternative angles, building exteriors |
| **Animated elements** | Motion graphics beyond simple highlights |

**Rule:** Add if time permits. Do not delay for these.

## DOCUMENT READOUT EXECUTION RULES

### Display Timing

| Element | Duration | Notes |
|---------|----------|-------|
| Document appears | 0.5 sec before readout begins | Let viewer register it |
| Hold during quote | Full duration of spoken quote | Never cut mid-quote |
| Post-quote hold | 2-3 seconds | Let it land |
| Key moment hold | 6-8 seconds | For smoking gun only |

### Framing

| Rule | Reason |
|------|--------|
| Fill 70-80% of frame | Readable on mobile |
| Highlight quoted text | Yellow or white box |
| Dim non-quoted text | 40% opacity |
| Show source attribution | Bottom corner, small text |

### Speed

| Readout Type | Pace |
|--------------|------|
| Normal quote | Match natural speaking pace |
| Damning evidence | Slow by 15% |
| Technical/legal | Normal, pause after |

### Emphasis

| Marker in Script | Edit Instruction |
|------------------|------------------|
| **Bold text** | Highlight that phrase on document |
| `[PRIMARY SOURCE READOUT]` | Full-screen document, slow delivery |
| `[KEY MOMENT]` | Hold 8 seconds, cut to camera reaction after |

## MAP USAGE RULES

### When Maps Appear

| Trigger | Map Type |
|---------|----------|
| Opening geographic hook | Wide → zoom to subject |
| "Here" / "This is where" | Highlight specific location |
| Territorial comparison | Split-screen or overlay |
| Historical boundary | Period-appropriate styling |

### When Maps Must Disappear

| Trigger | Action |
|---------|--------|
| Quote begins | Cut to document or camera |
| Emotional/argumentative point | Return to on-camera |
| Steelman section | On-camera only (credibility) |
| Conclusion | On-camera for authority |

### Map Animation Rules

| Animation | Use For | Avoid |
|-----------|---------|-------|
| Slow zoom | Opening hook | Fast zooms |
| Highlight pulse | "This area" callouts | Constant animation |
| Fade between eras | Historical comparison | Hard cuts between maps |
| Border draw | Boundary changes | Complex multi-layer reveals |

## ON-CAMERA DELIVERY GUIDANCE

### Tone Shifts by Section

| Section Type | Tone | Physical |
|--------------|------|----------|
| Opening hook | Direct, confident | Slight forward lean |
| Evidence presentation | Measured, authoritative | Neutral posture |
| Primary source readout | Slower, deliberate | Minimal movement |
| Steelman (counter-argument) | Respectful, fair | Open hands gesture |
| "That's not my interpretation" | Firm but not aggressive | Brief pause before |
| Conclusion | Measured, leaves question | Slight pause at end |

### Pause Markers

| Script Element | Pause Duration |
|----------------|----------------|
| After declarative fragment | 0.5-1 second |
| Before "But here's the thing" | 1 second |
| After smoking gun quote | 2 seconds |
| Before return-to-extremes | 1 second |
| Final line | 2 seconds before end card |

### Emphasis Words

| Word/Phrase | Delivery |
|-------------|----------|
| "Not" (in fragments) | Slight stress |
| Numbers (35, 1960, etc.) | Clear enunciation |
| "Restored" vs "declared" | Stress the contrast |
| "That's not my interpretation" | Even, factual tone |

## COMMON EXECUTION FAILURE MODES

### Filming Failures

| Failure | Result | Prevention |
|---------|--------|------------|
| Racing through readouts | Evidence doesn't land | Time each readout beforehand |
| Apologetic tone on caveats | Undermines authority | Caveats are confidence, not weakness |
| Over-dramatic on evidence | Comes across as partisan | Let documents carry weight |
| Monotone steelman | Sounds insincere | Genuine respect in voice |
| Rushing conclusion | No landing | Film conclusion separately, fresh |

### Editing Failures

| Failure | Result | Prevention |
|---------|--------|------------|
| Document appears late | Viewer misses context | 0.5 sec lead-in rule |
| Cutting during quote | Jarring, unprofessional | Hold through full quote |
| Map stays too long | Becomes wallpaper | Cut at natural break |
| No breathing room | Exhausting to watch | Honor pause markers |
| Burying key moment | Payoff lost | 8-second hold, camera reaction |

### Authority Failures

| Failure | Result | Prevention |
|---------|--------|------------|
| "I think maybe..." hedging | Sounds unsure | Use author-style caveats instead |
| Skipping "I went to documents" | Loses research credibility | Deliver authority marker fully |
| Reading quotes too fast | Evidence doesn't register | Slow by 15% on quotes |
| Aggressive on opponent's view | Looks partisan | Steelman with genuine respect |
