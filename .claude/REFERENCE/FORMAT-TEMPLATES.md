## Purpose

This document defines 5 repeatable content format templates that differentiate History vs Hype from all competitors.

**Core Differentiator:** Showing primary sources ON SCREEN in structured, repeatable formats (not just narrating them).

---

## When to Use Format Templates

**Before writing ANY script, ask:**
1. Does this topic fit a format template?
2. Which template best serves the evidence?
3. Can this become part of a series?

**Preference:** Use format templates when possible (builds series identity, viewer expectations, production efficiency).

**Default:** If no template fits, write custom structure (quality > template adherence).

## Template #1: BOTH EXTREMES ARE WRONG ⭐ **PRIMARY SERIES**

### When to Use:
- Topic has polarized online debate
- Two opposing claims about same historical event
- Both sides cite "history" but get it wrong
- Primary sources contradict BOTH narratives

### Structure (9-11 minutes):

**ACT 1: THE FALSE BINARY (2 min)**
```
OPENING HOOK (15 sec):
"Online, people argue about [TOPIC]. One side says [EXTREME A]. The other says [EXTREME B]. I looked at the primary sources. They're both wrong."

SHOW POLARIZATION (30 sec):
[Screenshots of social media, YouTube comments, articles from BOTH perspectives]

STAKES (1 min):
- Why does this debate matter?
- What modern consequences?
- Transition: "Let's look at what the sources actually say."
```

**ACT 2: DEBUNK EXTREME A (3-4 min)**
```
STATE THE CLAIM:
"Extreme A says: [EXACT CLAIM WITH SOURCE]"

SHOW PRIMARY SOURCE #1:
[Display document on screen]
Citation: [Author, Title, Page]
Quote: "[EXACT TEXT]"

EXPLAIN DISCREPANCY:
Why does claim contradict source?
Misquote? Anachronism? Motivated reasoning?

ACADEMIC QUOTE:
"According to [HISTORIAN] in [BOOK], p. [X]: '[DEBUNK QUOTE]'"
```

**ACT 3: DEBUNK EXTREME B (3-4 min)**
```
[Same structure as Act 2, but for opposite extreme]
```

**ACT 4: THE ACTUAL TRUTH (2-3 min)**
```
ACADEMIC CONSENSUS:
"So what DO the sources show? [SUMMARY]"

SHOW PRIMARY SOURCES:
[Timeline, map, or evidence compilation]

WHY IT'S MORE COMPLEX:
"The real story isn't [DRAMATIC VILLAIN]. It's [SYSTEMIC CAUSE]."

KRAUT CAUSAL CHAIN:
"[X] happened, which meant [Y], consequently [Z]..."

MODERN LESSON:
Why should viewers care about accurate history?
```

### Series Branding:
- **Title Formula:** "[Topic]: Both Extremes Are Wrong"
- **Thumbnail:** Red (left) vs Blue (right) with X through both
- **Intro:** "Welcome to Both Extremes Are Wrong, where we check what primary sources ACTUALLY say"
- **Hashtag:** #BothExtremesWrong

### Episode Examples:
1. Library of Alexandria: Both Extremes Are Wrong
2. Medieval Flat Earth: Both Extremes Are Wrong ← **NEXT WEEK**
3. Crusades Defensive War: Both Extremes Are Wrong
4. Columbus Genocide Claims: Both Extremes Are Wrong
5. Dark Ages Literacy: Both Extremes Are Wrong

## Template #2: DOCUMENT SHOWDOWN

### When to Use:
- Two sources/documents cited for opposite claims
- Treaty exists in multiple versions (English vs Spanish)
- Historical text vs modern interpretation
- Can display BOTH documents side-by-side

### Structure (10-12 minutes):

**ACT 1: THE COMPETING CLAIMS (2 min)**
```
OPENING HOOK (15 sec):
"[Group A] says [historical claim]. [Group B] says the OPPOSITE. So I got both primary sources. Let's put them on screen at the same time."

VISUAL SETUP:
Split screen: Document A (left), Document B (right)
Both visible throughout video
```

**ACT 2: LINE-BY-LINE COMPARISON (6-7 min)**
```
REPEAT FOR 3-5 KEY PASSAGES:

1. Highlight passage in Document A
2. Highlight corresponding passage in Document B
3. Zoom in to exact wording
4. Talking head explains what each says
5. Show contradiction/agreement

COLOR CODE:
- Green = where they agree
- Yellow = key contradictions
- Red = major discrepancy
```

**ACT 3: THE VERDICT (2-3 min)**
```
- What does evidence actually show?
- Why do both sides misrepresent?
- Modern implications
```

### Visual Requirements:
- PDF scans of both documents (high-res)
- Split-screen template in editing
- Text highlighting/zooming effects
- Side-by-side display THROUGHOUT

### Episode Examples:
1. Confederate Constitution vs Cornerstone Speech
2. 1859 Belize Treaty: English vs Spanish
3. Medieval Texts: Bede vs Flat-Earth Claims
4. ICJ Ruling Text vs Media Headlines

## Template #3: TREATY AUTOPSY

### When to Use:
- Territorial dispute involves specific treaty
- ICJ case pending or recently decided
- Legal document creates modern conflict
- Can access treaty full text

### Structure (12-15 minutes):

**ACT 1: THE MODERN DISPUTE (2 min)**
```
OPENING HOOK (15 sec):
"[Country A] and [Country B] are fighting over [territory]. The dispute comes down to ONE treaty from [year]. So I got the treaty. Let's read it. Every word."

STAKES:
- Modern conflict (ICJ case, military, diplomatic)
- Why treaty text matters RIGHT NOW
```

**ACT 2: THE TREATY TEXT - SECTION BY SECTION (8-10 min)**
```
FOR EACH ARTICLE:

1. Show article in full on screen
2. Read it out loud
3. Highlight key phrases (yellow)
4. Explain legal meaning
5. Show map of what article establishes
6. Note ambiguities (red)

COLOR CODE:
- Yellow = key binding phrases
- Red = ambiguous language
- Green = definitions/clarifications
- Blue = cross-references
```

**ACT 3: WHY THIS MATTERS TODAY (3-4 min)**
```
- How do lawyers argue about this text?
- ICJ precedent on similar ambiguities?
- Predicted outcome
```

### Episode Examples:
1. Belize-Honduras 1930 Exchange (ICJ pending)
2. 1859 Belize-Guatemala Treaty
3. 1959 Antarctic Treaty Article IV
4. Sykes-Picot Agreement Line-by-Line

## Template #4: THE MAP THEY IGNORED

### When to Use:
- Alternative borders were REALLY proposed (documented)
- Commission report, treaty draft exists
- Can show current borders vs alternative
- NOT speculation - must have PRIMARY SOURCE

### Structure (10-12 minutes):

**ACT 1: THE BORDER WE KNOW (2 min)**
```
OPENING HOOK (15 sec):
"This is [region]'s border today. But in [year], there was a different map—proposed by [source]. Here's what the world would look like if they'd used that map instead."

VISUAL:
- Show current borders
- Tease alternative briefly
```

**ACT 2: THE ALTERNATIVE MAP (5-6 min)**
```
1. Show actual historical document (map, report, proposal)
2. Establish provenance (who made? when? why?)
3. Display alternative borders on screen
4. Compare to current (side-by-side)
5. Explain differences

CRITICAL: Must have documentary evidence, NOT speculation
```

**ACT 3: WHAT WOULD BE DIFFERENT? (3-4 min)**
```
- Which conflicts might not exist?
- Why was alternative rejected?
- Stakeholder quotes from time period

AVOID: Speculation
FOCUS: What document proposed (fact), why rejected (documented)
```

### Episode Examples:
1. King-Crane Commission 1919 Middle East Map
2. India-Pakistan Boundary Commission Original Line
3. 1947 UN Partition vs What Happened
4. Casement Report Africa Borders

## Template #5: SAME DAY, DIFFERENT WAR

### When to Use:
- Major war with multiple theaters
- Significant date (D-Day, Pearl Harbor, etc.)
- Can access dated primary sources
- Educational value of simultaneity

### Structure (8-10 minutes):

**ACT 1: THE DATE (30 sec)**
```
OPENING HOOK (15 sec):
"June 6, 1944. Everyone knows D-Day. But do you know what was happening in the Pacific that same day? Or the Eastern Front? Let's look at the SAME date across three different theaters."

VISUAL: Three-way split screen tease
```

**ACT 2-4: THEATER 1, 2, 3 (2-3 min each)**
```
FOR EACH THEATER:

- What happened on this date?
- Show primary sources (war diaries, dispatches)
- Key statistics (casualties, territory, units)
- Maps showing theater

SAME FORMAT FOR ALL THREE
```

**ACT 5: THE COMPARISON (2 min)**
```
- What do we learn by seeing side-by-side?
- Resource allocation implications
- Relative scale/intensity
- Command decisions with full context
```

### Episode Examples:
1. June 6, 1944: D-Day Across All Theaters
2. December 7, 1941: Pearl Harbor, Manila, Wake
3. September 1, 1939: Poland, Britain, Far East

## Format Selection Guide

| If Topic Has... | Use Template |
|-----------------|--------------|
| Two polarized online claims | **BOTH EXTREMES ARE WRONG** |
| Two competing documents | **DOCUMENT SHOWDOWN** |
| Legal treaty dispute | **TREATY AUTOPSY** |
| Documented alternative borders | **THE MAP THEY IGNORED** |
| Multiple simultaneous theaters | **SAME DAY DIFFERENT WAR** |
| None of the above | Custom structure |

## Common Mistakes to Avoid

❌ **Forcing template when it doesn't fit**
✅ Custom structure better than poor template fit

❌ **Narrating sources instead of showing them**
✅ All templates require ON-SCREEN document display

❌ **Skipping series branding elements**
✅ Consistent intro/outro builds series recognition

❌ **Using template structure loosely**
✅ Follow Act structure exactly (creates viewer expectation)

## Series Management

**Active Series:**
- **BOTH EXTREMES ARE WRONG** - Primary series, launch first
- **DOCUMENT SHOWDOWN** - Secondary series, add after 5 "Both Extremes" episodes

**Experimental:**
- **TREATY AUTOPSY** - Use for ICJ cases only
- **THE MAP THEY IGNORED** - Quarterly specials (high production)
- **SAME DAY DIFFERENT WAR** - WWII content (semi-annual)

**Episode Tracking:**
Maintain playlist order, reference previous episodes in series

**Intro Consistency:**
ALWAYS use same intro for each series (builds brand recognition)
