# Post-Script Fact-Check Template

**PURPOSE:** Mandatory verification checklist to run BEFORE validating or filming any script.

**WHY THIS EXISTS:** To prevent errors like "Guatemala claims this entire country" (when they claim ~50%) from reaching production.

**TIME REQUIRED:** 15-20 minutes

**WHEN TO USE:** Immediately after completing FINAL-SCRIPT.md, before running VidIQ validation or filming.

---

## INSTRUCTIONS

Copy this file to your project folder as `POST-SCRIPT-FACT-CHECK.md` and complete all 6 steps:

```bash
# Copy template to project
cp .claude/POST-SCRIPT-FACT-CHECK-TEMPLATE.md video-projects/_IN_PRODUCTION/[project-name]/POST-SCRIPT-FACT-CHECK.md
```

Then work through each step, checking boxes as you verify.

---

## STEP 3: Flag Absolute Language for Verification (2 min)

**Task:** Find words that make absolute claims, verify each can be proven.

**Trigger words:**
- "All," "entire," "every," "never," "always," "none"
- "Completely," "totally," "fully," "entirely"
- "Proves," "definitively shows," "confirms beyond doubt"

### Absolute Language Audit:

**Search script for trigger words, list each instance:**

1. **"[Trigger word]" in context:** _________________
   - Can this be verified with primary source? Yes / No
   - Source: _________________
   - Should this be softened? Yes / No
   - Revised to: _________________

2. **"[Trigger word]" in context:** _________________
   - Verified? Yes / No
   - Source: _________________
   - Revised to: _________________

3. **"[Trigger word]" in context:** _________________
   - Verified? Yes / No
   - Source: _________________
   - Revised to: _________________

*[Continue for all instances]*

### Common Softening Patterns:
- ❌ "All historians agree" → ✅ "Scholarly consensus holds" (cite 3+ historians)
- ❌ "Never happened" → ✅ "No documented evidence of"
- ❌ "Entire country" → ✅ "Approximately half the country (~48%)"
- ❌ "Always rules this way" → ✅ "Pattern across cases (cite 3 examples)"
