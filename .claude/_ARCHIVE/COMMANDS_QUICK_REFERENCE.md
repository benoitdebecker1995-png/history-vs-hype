# COMMANDS QUICK REFERENCE
## Every Command & When to Use It

---

## CORE PRODUCTION WORKFLOW

### `/script`
**When:** After you've done NotebookLM research
**What it does:** Generates video script with visual staging notes
**Input needed:** Topic, thesis, NotebookLM research output
**Output:** Full production script with inline visuals

### `/edit-guide`
**When:** After you've filmed A-roll
**What it does:** Creates shot-by-shot editing instructions
**Input needed:** A-roll transcript with timestamps
**Output:** Detailed editing guide with visual staging

### `/youtube-metadata`
**When:** After video is edited
**What it does:** Generates title, description, tags, timestamps
**Input needed:** Final script, video runtime
**Output:** Copy-paste ready YouTube metadata

---

## QUALITY CONTROL (Optional)

### `/review-script`
**When:** Before filming (optional quality check)
**What it does:** Acts as critical viewer, finds credibility issues
**Input needed:** Draft script
**Output:** Report with 🔴 Major Problems, 🟡 Moderate Issues, 🟢 What Works

### `/fact-check`
**When:** Before filming (optional verification)
**What it does:** Systematic 3-tier fact-checking
**Input needed:** Script file
**Output:** Tier 1/2/3 fact-check reports with ✅ VERIFIED / ❌ ERROR / ⚠️ UNCERTAIN

---

## HELPER TOOLS

### `/format-sources`
**When:** Preparing video description
**What it does:** Organizes source list, verifies URLs
**Input needed:** List of sources from script
**Output:** Formatted source list by category with verified URLs

### `/notebooklm-prompts`
**When:** Before research phase
**What it does:** Generates custom NotebookLM research prompts
**Input needed:** Topic, thesis, research questions
**Output:** 8-11 copy-paste ready prompts for NotebookLM

---

## COMMAND COMPARISON

| Command | Required or Optional | When in Workflow | Time Needed |
|---------|---------------------|------------------|-------------|
| `/script` | **REQUIRED** | After research | 10-15 min |
| `/edit-guide` | **REQUIRED** | After filming | 5-10 min |
| `/youtube-metadata` | **REQUIRED** | After editing | 5 min |
| `/review-script` | Optional | Before filming | 10-15 min |
| `/fact-check` | Optional | Before filming | 20-30 min |
| `/format-sources` | Helpful | Before publishing | 5 min |
| `/notebooklm-prompts` | Helpful | Before research | 10 min |

---

## MINIMAL WORKFLOW (When Overwhelmed)

```
1. /script          (Generate script)
2. Film it
3. /edit-guide      (Get editing instructions)
4. Edit video
5. /youtube-metadata (Get title/description/tags)
6. Publish
```

**That's it.** Skip the optional steps when you're short on time.

---

## FULL WORKFLOW (When You Have Time)

```
1. /notebooklm-prompts  (Get research prompts)
2. Do NotebookLM research
3. /script              (Generate script)
4. /review-script       (Check for issues)
5. /fact-check          (Verify claims)
6. Fix issues
7. Film
8. /edit-guide          (Get editing guide)
9. Edit video
10. /format-sources     (Organize source list)
11. /youtube-metadata   (Get metadata)
12. Publish
```

---

## WHEN TO USE WHICH COMMAND

**"I need a script"**
→ `/script`

**"I filmed and need editing help"**
→ `/edit-guide`

**"Is my script credible?"**
→ `/review-script`

**"Are my facts accurate?"**
→ `/fact-check`

**"I need title and description"**
→ `/youtube-metadata`

**"My source list is messy"**
→ `/format-sources`

**"I need research prompts"**
→ `/notebooklm-prompts`

---

## QUICK TIPS

✅ **Always required:** /script, /edit-guide, /youtube-metadata
⭐ **High value optional:** /review-script (prevents credibility issues)
⏱️ **Time-saver:** /format-sources (makes description easier)
📋 **Research helper:** /notebooklm-prompts (organizes research phase)
🔍 **Paranoid mode:** /fact-check (when stakes are high)

---

**Remember:** You can skip optional steps and still make great videos. Don't let perfectionism block progress.
