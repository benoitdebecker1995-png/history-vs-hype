## Archived Agents (Future)

Agents that are not frequently used can be moved here to save tokens.

**Archived agents:**
- Still functional and can be used if needed
- Not loaded by default (saves tokens)
- Can be moved back to `/agents/` anytime

**To use an archived agent:**
1. Move the file from `_ARCHIVE/` to `agents/`
2. Use as normal

**Archiving criteria:**
- Large file size (600+ lines)
- Infrequent use
- Specialized workflows

---

**Note:** Archiving reduces token usage for common tasks while keeping files accessible for reference.
