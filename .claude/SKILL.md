# Documentary Editing Guide Skill

Creates comprehensive, shot-by-shot editing guides for documentary and educational videos using the hybrid talking-head + B-roll methodology.

## When to Use This Skill

Trigger this skill when users need:
- Shot-by-shot editing breakdowns for documentary videos
- Strategic decisions on talking head vs. B-roll placement
- Editing guides based on transcripts or scripts
- Production timelines and asset lists for educational content
- Guidance on visual rhythm and pacing for fact-based videos

## Core Methodology: Hybrid Talking-Head + B-Roll

### The Philosophy

**Default: Host on camera.** Cut away ONLY when B-roll strengthens the argument.

This approach:
- Establishes host as authority figure
- B-roll = EVIDENCE, not decoration
- Fits "primary sources" brand for educational content
- Creates trust through visible research

### When to Stay on Talking Head

- Making logical arguments or interpretations
- Explaining causation and motivations
- Connecting historical events or facts
- Asking rhetorical questions
- Providing analysis or synthesis
- All transitions between topics
- Emotional emphasis or moral judgments
- Complex explanations that need human connection

### When to Cut to B-Roll

**Only when visuals add necessary information:**

1. **Primary sources** - Treaties, documents, official records that prove claims
2. **Geographic evidence** - Maps showing boundaries, territories, movements
3. **Visual evidence** - Historical photos, portraits of key figures
4. **Current footage** - Modern situations being discussed
5. **Data visualization** - Statistics that need graphical representation

## Guide Structure

Every editing guide should contain these sections in order:

### 1. Format Overview
- Brief explanation of hybrid approach
- Estimated talking head vs. B-roll ratio
- Visual style notes specific to this video

### 2. Shot-by-Shot Breakdown

Format each shot entry as:

```
**[TIMESTAMP]** "[EXACT QUOTE FROM SCRIPT]"
- **SHOW:** [What should be on screen]
- **WHY:** [Strategic reasoning for this decision]
- **OPTIONAL:** [Additional notes: text overlays, transitions, pacing]
```

**Critical rules for shot entries:**
- Use EXACT timestamps from transcript/subtitle file
- Quote the actual words being said
- Always include both SHOW and WHY
- WHY should explain the strategic reasoning, not just describe
- Group related shots into meaningful sections (2-5 minutes each)

### 3. Visual Type Breakdown

After shot-by-shot section, provide:
- Percentage breakdown by visual type (talking head, maps, documents, etc.)
- Estimated quantities needed for each type
- Priority levels (essential vs. nice-to-have)

### 4. Must-Nail Moments

List 5-10 critical moments that need:
- Multiple takes
- Best performance/delivery
- Perfect timing
- Specific emotional tone

### 5. Visual Rhythm Strategy

Provide typical patterns like:
```
Typical 30-second pattern:
1. Host on camera (12 sec) - making argument
2. Cut to evidence (6 sec) - map/document  
3. Back to host (8 sec) - interpreting evidence
4. Cut to evidence (4 sec) - portrait/image
5. Back to host - transition
```

### 6. Asset Priority List

Organize needed B-roll into:
- **ESSENTIAL** (Must have - video doesn't work without these)
- **NICE TO HAVE** (Improves quality but not critical)
- **NOT NEEDED** (Skip these - common mistakes)

### 7. Production Timeline

Break down estimated hours for:
- Phase 1: Film talking head
- Phase 2: Edit talking head base
- Phase 3: Add strategic B-roll
- Phase 4: Polish (text, music, color)
- TOTAL time estimate

## Writing Style Guidelines

### Tone
- Direct and actionable
- Strategic, not just descriptive
- Assume competent editor who needs decision-making guidance
- Conversational but professional

### Language Patterns
- Use "**CRITICAL:**" for make-or-break elements
- Use "**WHY:**" to always explain strategic reasoning
- Use "Optional:" for nice-to-have elements
- Use imperative voice ("Show," "Cut to," "Hold for")

## Common Patterns to Apply

### The Hook (First 30 seconds)
- Fast pacing
- Immediate visual or conceptual hook
- Question or mystery setup
- Stakes established quickly

### Emotional Peaks
- Stay on host for maximum impact
- Longer takes, fewer cuts
- Strategic pauses marked in guide
- Build-up noted ("This is THE moment")

### Complex Explanations
- Break into multiple camera angles if host-only
- OR intercut with explanatory graphics/maps
- Mark where audience might get lost
- Suggest pattern interrupts

### Evidence Presentation
- Show document/source briefly (5-8 seconds max)
- Zoom to specific relevant text
- Back to host immediately for interpretation
- Never linger on text viewers can't fully read

## Quality Checklist

Before finalizing any editing guide, verify:

- [ ] Every timestamp is accurate to source material
- [ ] Every shot has both SHOW and WHY
- [ ] Strategic reasoning is clear, not just descriptive
- [ ] Must-nail moments identified
- [ ] Asset priority list is realistic
- [ ] Production timeline is honest
- [ ] Visual rhythm patterns are explained
- [ ] Guide is scannable (good headers, formatting)
- [ ] Both editorial and practical guidance included

## Adapting for Different Video Types

### Short-Form (Under 5 minutes)
- Simpler shot breakdown
- Fewer B-roll categories
- Faster pacing notes
- Higher talking head ratio

### Long-Form (10+ minutes)
- Chapter-based organization
- Pattern breaks every 3-4 minutes
- More detailed asset lists
- Pacing variation strategy

### Interview-Style
- Multiple camera angles for host
- Guest footage considerations
- B-roll to cover edits
- Rhythm between speakers
