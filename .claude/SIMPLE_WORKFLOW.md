# SIMPLE VIDEO WORKFLOW
## The Only Guide You Need

When you're ready to make a video, follow these steps. Nothing else.

---

## PHASE 1: RESEARCH & SCRIPT

**What you do:**
1. Research topic in NotebookLM
2. Run `/script` command
3. Read the generated script

**Stop here if:** Script needs major changes. Otherwise, move to filming.

**Time:** 2-4 hours

---

## PHASE 2: FILM

**What you do:**
1. Open autocue script file
2. Film yourself reading it
3. Multiple takes of hard sections

**That's it.** Don't fact-check yet. Don't review yet. Just film.

**Time:** 1-2 hours

---

## PHASE 3: EDIT

**What you do:**
1. Export A-roll transcript with timestamps
2. Run `/edit-guide` with your transcript
3. Follow the shot-by-shot suggestions
4. Edit in DaVinci Resolve

**Time:** 1-2 days

---

## PHASE 4: PUBLISH

**What you do:**
1. Run `/youtube-metadata` with your script
2. Copy title, description, tags
3. Upload video
4. Done

**Time:** 30 minutes

---

## OPTIONAL QUALITY CHECKS

**Only do these if you have time:**

### Before Filming (Optional):
- `/review-script` - Check for credibility issues
- `/fact-check` - Verify all claims

### After Editing (Optional):
- Watch full video
- Check retention at 0:15, 2:00, 5:00
- Adjust if needed

---

## THE RULE

**If a step feels overwhelming, skip it and move forward.**

Better to publish an 80% video than never finish a 100% video.

---

## COMMANDS CHEAT SHEET

```
/script          → Generate video script
/edit-guide      → Get editing instructions
/youtube-metadata → Get title/description/tags
/review-script   → (Optional) Check credibility
/fact-check      → (Optional) Verify claims
```

---

**That's the whole workflow. Four phases. Done.**
