## How to Use This File

1. Before publishing, log your title/thumbnail variants
2. After 48-72 hours, record CTR for each variant
3. After 7 days, record final CTR
4. Analyze patterns across multiple videos
5. Update "What Works" section with insights

---

## Testing Methods

### YouTube A/B Testing (If Available)
- YouTube sometimes offers native A/B testing
- If available, use it - most reliable data

### Manual A/B Testing
1. Publish with Variant A
2. After 48 hours, note CTR + impressions
3. Switch to Variant B
4. After 48 hours, note CTR + impressions
5. Compare at equal impression levels
6. Keep the winner

### VidIQ Title Testing
1. Enter 3-4 title variants
2. Compare VidIQ scores
3. Pick highest score (80+)
4. Log results below

#### VidIQ Validated Thumbnail Insights (2025-12-04)

**⚠️ MAP THUMBNAILS OUTPERFORM FACE THUMBNAILS BY 26x**

| Thumbnail Type | VidIQ Score | Actual Views | Result |
|----------------|-------------|--------------|--------|
| Map-focused (Essequibo) | 72 | 1,905 | Best performer |
| Face-focused | 85+ | ~73 | Baseline |

**Key Insight:** VidIQ scores optimize for generic YouTube. This channel's niche (territorial disputes, documentary style) performs better with map/evidence thumbnails despite VidIQ preferring faces.

**Recommendation:**
- Default to map/evidence thumbnails
- Ignore VidIQ thumbnail score for this channel
- Test face thumbnails only for political figure fact-checks
