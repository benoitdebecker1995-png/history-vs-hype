# Archived Projects

**Purpose:** Completed or cancelled video projects

## What Goes Here:

### 1. Published Videos:
- Completed projects after video is published
- Keep for reference and future updates
- Include final script, analytics, and lessons learned

### 2. Cancelled Projects:
- Topics that were researched but abandoned
- Scripts that didn't meet quality standards
- Projects postponed indefinitely

### 3. Superseded Projects:
- Old versions replaced by better approaches
- Research compilations absorbed into other projects

## Archive Structure:

### For Published Videos:
```
topic-name-2025/
├── PUBLISHED-INFO.md        # Date, URL, initial analytics
├── script-FINAL.md
├── sources.md
├── analytics-snapshot.md    # Performance data
└── lessons-learned.md       # What worked, what didn't
```

### For Cancelled Projects:
```
topic-name-cancelled/
├── CANCELLATION-REASON.md   # Why it was abandoned
├── research-summary.md      # What was learned
└── [partial work]
```

## Retention Policy:
- Published videos: Keep permanently (valuable reference)
- Cancelled projects: Review annually, delete if no longer useful
- Old drafts: Consolidate into single archive file

## When to Archive:
- Video is published ✅
- Project definitively cancelled ❌
- Better version of topic completed 🔄
