# Ready to Film

**Purpose:** Completed scripts ready for filming and production

## What Goes Here:
- Scripts that are 100% fact-checked
- Projects with B-roll checklist completed
- Videos ready to be filmed at any time
- No blocking issues remaining

## Requirements to Enter This Folder:
- [ ] Script finalized and approved
- [ ] All claims fact-checked (2+ sources each)
- [ ] Retention prediction ≥ 40% (if analyzed)
- [ ] B-roll checklist created
- [ ] Sources formatted for video description
- [ ] Modern relevance hooks confirmed
- [ ] Pre-filming checklist completed

## When to Move:
- **To _ARCHIVED**: After video is filmed, edited, and published

## Priority System:
Use numeric prefixes to indicate filming priority:
- `1-topic-name/` - High priority (film next)
- `2-topic-name/` - Medium priority
- `3-topic-name/` - Low priority / backup content

## Example Structure:
```
_READY_TO_FILM/
├── 1-sykes-picot-2025/
│   ├── script-FINAL.md
│   ├── B-ROLL-CHECKLIST.md
│   ├── sources-formatted.md
│   └── FACT-CHECK-VERIFIED.md
```
