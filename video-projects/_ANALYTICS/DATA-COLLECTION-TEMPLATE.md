## DATA QUALITY GUIDELINES

**Minimum required for useful analysis:**
- Views
- Retention %
- Topic type
- Modern hook (Yes/No)

**Nice to have:**
- CTR
- Traffic source
- Subs gained
- Specific notes on why it worked/failed

**Don't stress about:**
- Perfect accuracy (estimates okay)
- Complete data for every field
- Going through all 182 videos immediately

**Start with 20 videos (top 10 + bottom 10) for initial pattern detection.**

---

## WHEN TO SUBMIT DATA

**After Batch 1 (Top 10) complete:**
Submit to performance-analyzer agent for initial outlier DNA extraction

**After Batch 2 (Bottom 10) complete:**
Submit to performance-analyzer for pattern analysis (top vs. bottom comparison)

**After Batch 3 (Middle 20) complete:**
Submit for confidence-level increase and validation framework generation

**Ongoing:**
Add each new video published to track real-time pattern validation

---

## HOW TO SUBMIT TO AGENT

Once you've filled out 10+ videos, say:

"Analyze performance data: [paste the filled-out section]"

The performance-analyzer agent will:
1. Add to performance-database.md
2. Extract patterns
3. Update topic validation framework
4. Identify replication opportunities
