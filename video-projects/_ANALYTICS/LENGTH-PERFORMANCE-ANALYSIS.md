## RECOMMENDED TRAVEL PRODUCTION FORMAT

### Target: 6-8 Minutes (JD Vance Model)

**Structure:**
1. **Hook (0:00-0:45):** Modern relevance + stakes [~80 words]
2. **Payoff (0:45-1:30):** Strongest evidence FIRST [~80 words]
3. **Evidence (1:30-5:30):** 3-4 key claims with sources [~450 words]
4. **Modern connection (5:30-7:00):** Why this matters [~160 words]
5. **CTA (7:00-8:00):** Subscribe, sources [~50 words]

**Total: ~820 words @ 110 wpm = 7:27 target**
