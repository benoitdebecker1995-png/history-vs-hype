## TROUBLESHOOTING COMMON ISSUES

**"This talking head segment feels too long"**
→ Check if it's analytical/interpretive content. If yes, stay on camera. Only cut to B-roll if there's EVIDENCE to show.

**"I have great B-roll but it's not in the guide"**
→ Ask: Does this EVIDENCE prove a claim, show a primary source, or visualize data? If no = decoration. Don't use.

**"The timeline sequence is confusing"**
→ Slow the build. Give each date 3-4 seconds before adding next. Use color coding (blue = Christian, orange = pagan).

**"Latin text is hard to read"**
→ Increase font size. Use high contrast (dark text on light parchment). Add translation animation appearing below.

**"Running short on time"**
→ Trim talking head segments by 1-2 seconds each. DO NOT cut the timeline or quote cards.

**"Running long"**
→ Look for talking head segments where same point is made twice. The steelman section (Shot 28) can be trimmed if needed.
