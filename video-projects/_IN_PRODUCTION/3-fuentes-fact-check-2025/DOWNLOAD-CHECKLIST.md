## TROUBLESHOOTING

### "PDF download not available"
**Solution:** Use browser Print function → Save as PDF

**Steps:**
1. Open webpage
2. Press Ctrl+P (Windows) or Cmd+P (Mac)
3. Choose "Save as PDF" as printer
4. Click Save
5. Name file appropriately

---

### "Google Scholar article behind paywall"
**Solution:** Look for free alternatives

**Try:**
1. Look for [PDF] link on right side of search result
2. Check if institution offers free access
3. Search article title + "PDF" on regular Google
4. Use ResearchGate or Academia.edu for author-uploaded versions
5. Skip if not freely available (articles are optional)

---

### "Founders Online search not finding document"
**Solution:** Use Avalon Project backup URLs provided

**Avalon Project is more direct:**
- Avalon Law Yale: https://avalon.law.yale.edu/
