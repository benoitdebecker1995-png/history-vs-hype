### Asset 1: Fulcher of Chartres Jerusalem Quote

**Text to Display:**
> "About ten thousand were beheaded. Not one of them was allowed to live. They did not spare the women and children."

**Free Tool:** Canva (free account)
**Time:** 20 minutes

**Step-by-Step:**
1. Go to Canva.com (create free account if needed)
2. Search templates: "parchment quote" or "medieval text"
3. Choose template with aged paper background
4. Replace text with the quote above
5. Font recommendations (free in Canva):
   - "Libre Baskerville" (readable medieval style)
   - "Cinzel" (classical, elegant)
   - "Old Standard TT" (traditional)
6. Add at bottom: "Fulcher of Chartres (Eyewitness, 1099)"
7. Download as PNG (1920x1080)

**Alternative (Even Faster):**
1. Open PowerPoint/Google Slides
2. Insert shape: rectangle (fill with beige/tan color)
3. Add texture: Right-click shape → Format → Texture fill → "Parchment" or "Paper"
4. Add text box with quote
5. Font: Garamond, Georgia, or Times New Roman (classic)
6. Export as image

**File Name:** doc-01-fulcher-jerusalem.png

### Asset 6: Fourth Crusade Diversion Map

**Content:** Map showing intended route (to Egypt) vs. actual route (to Constantinople)

**Free Tool:** Google My Maps OR MapChart.net
**Time:** 45 minutes

**Google My Maps Method:**
1. Go to google.com/maps → Menu → "Your places" → "Maps" → "Create Map"
2. Add markers:
   - Venice (start)
   - Zara (first stop)
   - Constantinople (actual destination)
   - Alexandria, Egypt (intended destination - mark in different color)
3. Draw lines:
   - Solid line: Venice → Zara → Constantinople (red)
   - Dashed line: Venice → Egypt (gray/faded)
4. Add labels: "INTENDED" vs "ACTUAL"
5. Screenshot the map (use Windows Snipping Tool)

**MapChart.net Method:**
1. Go to mapchart.net
2. Choose "Europe" map
3. Color code:
   - Venice, Zara, Constantinople (red)
   - Egypt (gray)
4. Download as PNG
5. Add arrows in PowerPoint/Paint

**File Name:** map-04-fourth-crusade-route.png

## CANVA PRO TIPS (Free Account)

**Best Free Templates for This Project:**
1. Search "parchment" → aged paper backgrounds
2. Search "timeline" → horizontal timeline layouts
3. Search "quote" → text-focused designs
4. Search "comparison chart" → data tables

**Keyboard Shortcuts:**
- Ctrl+D: Duplicate element (fast way to copy your template)
- Ctrl+G: Group elements (keep things together)
- Ctrl+Shift+K: Turn text into link (if needed)

**Download Settings:**
- File Type: PNG (best for transparency)
- Quality: Standard (free accounts can't do high-quality, but standard is fine for 1080p video)
