## CANVA CREATION GUIDE

### Step 1: Setup
1. Go to [canva.com/youtube-thumbnails/templates/](https://www.canva.com/youtube-thumbnails/templates/)
2. Create blank design: **1280 x 720 pixels**
3. Background: **#1a1a2e** (dark navy) or pure black

### Step 2: Create Left Document (1832)
1. Add rectangle: 350 x 450 px
2. Fill: Parchment/cream color (#f5e6c8)
3. Add text inside:
   - "MIRARI VOS" (serif font, 28pt)
   - "1832" (bold, 40pt)
   - "Religious freedom is ABSURD" (highlight "ABSURD" in red)
4. Add subtle red overlay/tint (opacity 20%)
5. Position: Left third of canvas

### Step 3: Create Right Document (1965)
1. Duplicate left document
2. Change text:
   - "DIGNITATIS HUMANAE"
   - "1965"
   - "Right to religious liberty"
3. Change overlay to green tint (opacity 20%)
4. Position: Right third of canvas

### Step 4: Add Arrow
1. Elements → Shapes → Arrow
2. Style: Bold, white or yellow
3. Position: Between documents, pointing right
4. Symbolizes: Time passing, reversal

### Step 5: Add Main Text
1. Add text: `101 YEARS`
2. Font: **Bebas Neue** or **Impact**
3. Size: 120pt
4. Color: White with black outline (3px)
5. Position: Bottom center

### Step 6: Final Touches
1. Add subtle vignette (darker edges)
2. Optional: Small Vatican seal icon
3. **NO FACE** - evidence speaks for itself

### Step 7: Export
1. Download → PNG
2. Test at 100px width on phone
3. Can you read "101 YEARS" instantly? If yes, done.
