# Attribution Errors Log - Vance Part 2

## Date: November 7, 2025

### Critical Errors Found

#### Error 1: "Tools with Voice" Misattributed to Luther
**What the script says:**
- "Martin Luther calls slaves 'tools with voice.'"

**The problem:**
- This quote is from **Aristotle**, not Martin Luther
- Aristotle in *Politics* (Book I) described slaves as "animate tools" or "living instruments"
- This is a secondary source error - someone incorrectly attributed Aristotle's concept to Luther

**Where it appears:**
- VANCE_PART_2_AUTOCUE.md (line 106)
- VANCE_PART_2_EDITING_GUIDE.md (line 285-289)
- vance video 2.srt
- Multiple other script versions

**Severity:** HIGH - This is exactly the kind of error we fact-check others for

## Root Cause Analysis

### Why This Happened

1. **Secondary source contamination**: Likely relied on a source that misattributed Aristotle to Luther
2. **Insufficient primary source verification**: Didn't check Luther's actual writings
3. **No citation tracking**: No documentation of which specific work/page the quote comes from
4. **Time pressure**: May have accepted research at face value without deep verification

### What Should Have Been Done

**For Luther Quote:**
- [ ] Check: "Where did Luther write about slavery?"
- [ ] Find: Specific treatise/letter/sermon
- [ ] Verify: Actual quote in original work (or good translation)
- [ ] Document: Work title, year, page/section number
- [ ] Cross-check: Is this quote commonly misattributed?

**For Thornwell Quote:**
- [ ] Find: Specific sermon, book, or article
- [ ] Verify: Exact wording in original text
- [ ] Document: Full citation (title, date, page)
- [ ] Confirm: Is this paraphrase or direct quote?

## Lessons Learned

1. **Every quote needs a specific source** - not just "Luther said" but "Luther, in [Title], [Year], [Page]"
2. **Famous quotes are often misattributed** - especially cross-cultural attributions
3. **Secondary sources can be wrong** - always trace back to primary source when possible
4. **Vague attributions are red flags** - "1850s" instead of specific date suggests incomplete research
