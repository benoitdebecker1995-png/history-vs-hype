# In Production

**Purpose:** Active video projects currently being developed

## What Goes Here:
- Projects with research in progress
- Scripts being written or revised
- Projects in fact-checking phase
- Anything actively being worked on

## Status Indicators:
Each project folder should contain a `STATUS.md` file with:
- Current phase (Research / Scripting / Fact-checking / Revising)
- Next action required
- Blockers (if any)
- Target completion date

## When to Move:
- **To _READY_TO_FILM**: When script is final, fact-checked, and B-roll checklist complete
- **To _ARCHIVED**: If project is cancelled or postponed indefinitely

## Example Structure:
```
_IN_PRODUCTION/
├── topic-name-2025/
│   ├── STATUS.md
│   ├── research.md
│   ├── script-draft.md
│   └── sources.md
```
