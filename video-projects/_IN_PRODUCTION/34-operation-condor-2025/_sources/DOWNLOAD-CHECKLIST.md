## HOW TO DOWNLOAD NSA DOCUMENTS

1. **Visit the briefing book URL**
2. **Scroll down** to find document list
3. **Look for:** "Document 1:", "Document 2:", etc.
4. **Click** the PDF link for each document
5. **Save** to appropriate folder with descriptive name

**Naming convention:**
```
P1-Doc01-Condor-Founding-Agreement-1975.pdf
P1-Doc02-CIA-Cable-Second-Meeting-1976.pdf
P2-Doc01-Kissinger-Blocking-Order-Sept16-1976.pdf
```
