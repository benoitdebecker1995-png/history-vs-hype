## DAY 1: CRITICAL ASSETS (Film after these)

### Asset 1: Paraguay Location Map
**Type:** Geographic context map
**Tool:** MapChart.net (free, no signup required)
**Time:** 15 minutes

**Steps:**

1. Go to **mapchart.net** (no login needed)
2. Click **"Create map"** → Select **"Blank map"**
3. Select **"South America"** region
4. In the left sidebar, find **"Paraguay"**
5. Click Paraguay to highlight it (should turn red)
6. Find **"Asunción"** on the map and click it to add a marker
7. Under **"Edit"** panel:
   - Set title to: "Operation Condor Discovery Site"
   - Add description: "Asunción, Paraguay - Archives of Terror found December 22, 1992"
8. Color scheme: Keep red for Paraguay (high contrast)
9. Export as **PNG** (right-side menu, "Download image")
10. Save as: **`01-paraguay-map.png`**

**Visual Quality:**
- Minimum 1920px width
- Ensure Asunción is clearly marked
- Contrast should be clear (red on white background)

**Alternative:** If MapChart is slow, use Google My Maps:
- Go to **mymaps.google.com**
- Create new map
- Search "Paraguay"
- Mark Asunción with custom marker
- Export as image (right-click → download)

**File Location:** Save to `video-projects/34-operation-condor-2025/assets/maps/`

### Asset 3: Founding Charter Document (Template)
**Type:** Primary source document display
**Tool:** Canva (free account, canva.com)
**Time:** 20 minutes

**Steps:**

1. Go to **canva.com**, create free account (or log in)
2. Click **"Create a design"** → Search **"Document"** → Select **"A4 Document"**
3. Set background to **cream/off-white** (simulate aged paper)
   - Click background → Color → #F5F1E8 (cream color)
4. Add text box with title:
   - Text: **"Operation Condor Founding Charter"**
   - Font: Serif (Times, Garamond, Georgia)
   - Size: 48px, bold
   - Color: Dark brown or black
5. Add subtitle:
   - Text: **"November 28, 1975 - Santiago, Chile"**
   - Font: Same serif, 28px, italic
   - Color: Gray
6. Add document body text (leave most blank, add key details):
   - Text: **"Signed by intelligence chiefs from five nations"**
   - Font: 20px, serif
   - Add spacing (click-drag elements to show document authenticity)
7. Bottom footer:
   - Text: **"Source: Archives of Terror, Paraguay - Discovered 1992"**
   - Font: 12px, gray
8. Download as **PNG** (1920x1080 or higher)
9. Save as: **`03-founding-charter-template.png`**

**Visual Specifications:**
- Cream/aged paper background
- Readable at video resolution
- Margins on all sides (white space)
- Serif font (official/historical feel)

**Alternative:** Use Microsoft Word:
- Create blank document
- Set background to cream color (Format → Page Color)
- Type title and key text
- Print to PDF, then screenshot
- Crop and save as PNG

**File Location:** Save to `video-projects/34-operation-condor-2025/assets/documents/`

**Usage in Video:** Shot 10 (3:45-4:30) - display while narrating founding details

## PRO TIPS FOR FASTER CREATION

### Batch Workflow: Create All Text Assets in One Session

1. **Open all Canva designs at once** (use multiple browser tabs)
2. **Create a "master design"** in the first asset
   - Set color palette (dark navy, cream, accent color)
   - Set typography (serif for quotes, sans-serif for info)
3. **Copy this design** for subsequent assets
   - Just change text content, keep styling consistent
   - Saves 5-7 minutes per asset
4. **Use Canva templates** as starting points
   - Search "quote card," "statistics," "timeline"
   - Customize template instead of building from scratch
   - Saves 10+ minutes per asset

### Keyboard Shortcuts (Canva)

- **Ctrl+D** = Duplicate selected element
- **Ctrl+A** = Select all
- **Ctrl+Z** = Undo
- **Shift+Click** = Multi-select
- **Double-click text** = Edit immediately

### Time-Saving Techniques

1. **Reuse designs:**
   - Create one "quote card" template
   - Duplicate it
   - Change quote text
   - Done in 3 minutes instead of 18

2. **Match sizing:**
   - First asset: Set width to 1920px
   - All subsequent assets: Use same width
   - Ensures consistency, no resizing needed

3. **Download in bulk:**
   - Create 3-4 assets, download all at once
   - Batch organization (select, right-click, download)
   - Saves 10 minutes on file management

## QUALITY STANDARDS FOR 1080P VIDEO

### Text Specifications

- **Minimum font size in video:** 24px
- **For readable subtitles:** 32px+
- **For quotes:** 48px+
- **For large stats:** 72px+

### Image Resolution

- **Minimum width:** 1920px (for 1080p video)
- **Canva default is good:** Most templates export at 1920+
- **Check before download:** If image looks small, increase canvas size

### File Formats

- **Export as PNG** (not JPG) for:
  - Text graphics
  - Documents
  - Quotes (sharp text quality)

- **Export as PNG or MP4** for animated graphics

### Color Contrast

- **Text on background:** Minimum 4.5:1 ratio (readable)
- **Dark text on light:** Always works
- **Light text on dark:** Always works
- **Avoid:** Light text on light, dark on dark

## TROUBLESHOOTING

**Canva is slow/loading issue:**
- Use Microsoft PowerPoint instead (local, no cloud dependence)
- Export same way: Screenshot → save as PNG

**MapChart not loading:**
- Use Google My Maps instead
- Or create in PowerPoint using text + shapes

**PNG file too large:**
- Right-click image in Canva download
- Select "Compress" or use online PNG compressor
- Canva usually optimizes automatically

**Text not readable in video:**
- Increase font size by 20-30%
- Add background color behind text if on complex background
- Test at actual video resolution (1080p)
