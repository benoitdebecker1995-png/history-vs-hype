## DOWNLOAD INSTRUCTIONS

### For Web Pages → PDF
1. Open URL in browser
2. Print → Save as PDF
3. Name format: `[Year]-[Document-Name].pdf`
4. Example: `1906-Persian-Constitution.pdf`

### For Archive.org
1. Click "PDF" download option
2. Or use "Download Options" → PDF
3. May need to create free account

### For State Department FRUS
1. Go to chapter list
2. Download each chapter as PDF
3. Or print full chapter to PDF
