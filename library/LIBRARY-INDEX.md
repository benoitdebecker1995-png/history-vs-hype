## MAINTENANCE

### Adding New Books:
1. Download to `Downloads/` folder
2. Identify category (crusades, colonialism, middle-east, etc.)
3. Rename to format: `Author - Title (Year).pdf`
4. Move to appropriate `by-topic/` folder
5. Add entry to this catalog (LIBRARY-INDEX.md)

### Starting New Project:
1. Create folder in `library/for-notebooklm/[project-name]/`
2. Copy (not move) relevant books from `by-topic/`
3. Upload to NotebookLM
4. Document in "Search by Project" section

### Catalog Updates:
- Add new books as you organize them
- Update "Projects Used" when books are referenced
- Keep "Total Books" count current

## FILE NAMING STANDARDS

**Books:**
```
Author - Title (Year).pdf
Barr - A Line in the Sand (2011).pdf
```

**Multi-author:**
```
LastName1 - Title (Year).pdf
Pigeaud - Africas Last Colonial Currency CFA Franc (2018).pdf
```

**Edited collections:**
```
Editor - Title (Year).pdf
Gabrieli - Arab Historians of the Crusades (2010).pdf
```

**No Z-Library, Anna's Archive, WeLib suffixes**
**No DOI numbers or hash codes**
**No compression artifacts (_compre, _compre-1)**
