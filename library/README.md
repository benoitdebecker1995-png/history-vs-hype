## FOR NOTEBOOKLM PROJECTS

**Workflow:**
1. Create project folder: `for-notebooklm/[project-name]/`
2. **Copy** (don't move) books from `by-topic/` folders
3. Upload copies to NotebookLM
4. Run research prompts
5. Keep original books in `by-topic/` for future projects

**Example:**
```
library/for-notebooklm/crusades-project/
├── Asbridge - The Crusades (2010).pdf
├── Hillenbrand - Islamic Perspectives (1999).pdf
└── Krey - Eyewitness Accounts (1921).pdf
```

## FILE NAMING STANDARD

**Books:**
```
Author - Title (Year).pdf
```

**Multi-author:**
```
FirstAuthor - Title (Year).pdf
```

**Edited collections:**
```
Editor - Title (Year).pdf
```

**NO:**
- Z-Library suffixes
- Anna's Archive metadata
- DOI/hash codes
- Compression artifacts
